## Fundamentos: Como os Algoritmos Funcionam

### A Verdade Sobre Algoritmos

Todos os algoritmos de redes sociais têm o **mesmo objetivo**: maximizar o tempo do usuário na plataforma. Eles fazem isso mostrando conteúdo que gera **engajamento**.

```
ALGORITMO = f(Engajamento, Relevância, Recência, Relacionamento)

Onde:
- Engajamento: Likes, comentários, compartilhamentos, salvamentos, tempo de visualização
- Relevância: Quão relacionado o conteúdo é com os interesses do usuário
- Recência: Quão recente é o post
- Relacionamento: Força da conexão entre criador e espectador
```

### Hierarquia de Sinais (Todos os Algoritmos)

```
TIER 1 - Sinais Fortes (Alto Peso)
├── Tempo de visualização / Watch time
├── Compartilhamentos (especialmente DM)
├── Salvamentos / Bookmarks
├── Comentários longos / replies
└── Cliques no perfil após ver conteúdo

TIER 2 - Sinais Médios
├── Likes / Reações
├── Comentários curtos
├── Follows após engajamento
└── Revisitas ao conteúdo

TIER 3 - Sinais Fracos
├── Impressões
├── Scroll past (negativo)
├── Hide / Not interested (muito negativo)
└── Unfollow após ver (muito negativo)
```

### O Conceito de "Velocity" (Velocidade Inicial)

```
JANELA CRÍTICA = Primeiros 30-60 minutos após publicação

Se engajamento alto na janela crítica:
  → Algoritmo distribui para mais pessoas
  → Cria efeito bola de neve
  → Potencial viral

Se engajamento baixo na janela crítica:
  → Algoritmo limita distribuição
  → Post "morre"
  → Alcance limitado a seguidores próximos
```

**Táticas para maximizar velocity:**
1. Publicar quando audiência está online
2. Responder comentários imediatamente (primeiros 15 min)
3. Usar Stories para direcionar ao post
4. Pedir engajamento específico no início
5. Criar "pod" orgânico com outros criadores

## Psicologia do Engajamento

### Por Que as Pessoas Engajam

Baseado em pesquisas de Jonah Berger (Contagious) e Robert Cialdini (Influence):

```
GATILHOS DE ENGAJAMENTO

1. SOCIAL CURRENCY (Moeda Social)
   "Compartilho porque me faz parecer inteligente/informado/divertido"
   → Conteúdo que faz a pessoa parecer bem
   → Dados exclusivos, insights únicos
   → Humor inteligente

2. TRIGGERS (Gatilhos)
   "Compartilho porque algo me lembrou disso"
   → Associar conteúdo a eventos frequentes
   → Segunda-feira, café da manhã, reunião
   → Criar associações mentais

3. EMOTION (Emoção)
   "Compartilho porque me fez sentir algo forte"
   → Emoções de alta excitação: Awe, raiva, ansiedade, diversão
   → EVITAR: Tristeza, contentamento (baixa excitação)
   → Quanto mais intensa a emoção, mais compartilhamentos

4. PUBLIC (Público)
   "Compartilho porque outros estão fazendo"
   → Prova social visível
   → Números de engajamento
   → Tendências e trends

5. PRACTICAL VALUE (Valor Prático)
   "Compartilho porque é útil"
   → Dicas, tutoriais, how-tos
   → Listas, checklists
   → Ferramentas, recursos

6. STORIES (Histórias)
   "Compartilho porque a história é boa"
   → Narrativas envolventes
   → Jornada do herói
   → Conflito e resolução
```

### Emoções que Geram Ação

```
MATRIZ DE EMOÇÕES (baseado em Berger, 2013)

                    Alta Excitação          Baixa Excitação
                   ─────────────────────   ─────────────────
Positivas         │ Awe (admiração)      │ Contentamento
(Compartilham)    │ Excitação            │ Relaxamento
                  │ Diversão/Humor       │
                   ─────────────────────   ─────────────────
Negativas         │ Ansiedade            │ Tristeza
(Não compartilham)│ Raiva                │ Desilusão
                  │ Medo (controlado)    │
                   ─────────────────────   ─────────────────

REGRA: Emoções de ALTA EXCITAÇÃO geram 3x mais compartilhamentos
```

### O Modelo "Pattern Interrupt"

```
ATENÇÃO = PADRÃO QUEBRADO

O cérebro ignora o previsível. Para capturar atenção:

1. DISRUPÇÃO VISUAL
   ├── Cores inesperadas
   ├── Movimento onde não esperado
   └── Contraste extremo

2. DISRUPÇÃO COGNITIVA
   ├── Afirmação contrária ao senso comum
   ├── Pergunta que não faz sentido imediato
   └── Paradoxo ou contradição

3. DISRUPÇÃO EMOCIONAL
   ├── Emoção inesperada
   ├── Humor onde não esperado
   └── Vulnerabilidade genuína
```

### Vieses Cognitivos no Engajamento

```
VIESES QUE USAMOS ESTRATEGICAMENTE

1. CURIOSITY GAP (Lacuna de Curiosidade)
   "O cérebro odeia informação incompleta"
   → "A técnica que 99% ignora..."
   → "O erro que todo mundo comete..."

2. LOSS AVERSION (Aversão à Perda)
   "Dor de perder > Prazer de ganhar"
   → "Você está perdendo..." > "Você pode ganhar..."
   → "Pare de fazer isso..."

3. SOCIAL PROOF (Prova Social)
   "Se outros fazem, deve ser bom"
   → Números, estatísticas, testemunhos
   → "Milhares já descobriram..."

4. AUTHORITY (Autoridade)
   "Especialistas sabem mais"
   → Credenciais, experiência
   → "Depois de 10 anos fazendo X..."

5. SCARCITY (Escassez)
   "Raro = valioso"
   → "Pouca gente sabe..."
   → "Informação restrita..."

6. RECIPROCITY (Reciprocidade)
   "Se você me dá, eu devo retribuir"
   → Dar valor primeiro
   → Engajamento gera engajamento
```

## Ciência da Viralidade

### O Que Faz Conteúdo Viralizar

Baseado em análises de milhões de posts virais:

```
FÓRMULA DA VIRALIDADE

V = (E × R × T) / F

Onde:
V = Potencial viral
E = Emoção evocada (0-10)
R = Relevância para audiência ampla (0-10)
T = Timing/Momento cultural (0-10)
F = Fricção para compartilhar (1-10)

Para maximizar V:
- Maximize E, R, T
- Minimize F (facilite o compartilhamento)
```

### Os 6 Tipos de Conteúdo Viral

```
TIPO 1: UTILITY (Utilidade)
├── Tutoriais, how-tos, hacks
├── Salvar para usar depois
└── Compartilhar para ajudar outros
Exemplo: "5 atalhos de Excel que economizam 2h/dia"

TIPO 2: ENTERTAINMENT (Entretenimento)
├── Humor, diversão, surpresa
├── Compartilhar para entreter outros
└── Conteúdo leve e consumível
Exemplo: Memes, vídeos engraçados, fails

TIPO 3: INSPIRATION (Inspiração)
├── Histórias de superação
├── Citações motivacionais contextualizadas
└── Transformações (antes/depois)
Exemplo: "Como fui de 0 a 100k seguidores"

TIPO 4: CONTROVERSY (Controvérsia)
├── Opiniões contrárias ao mainstream
├── Debates polarizados
└── "Hot takes" fundamentados
Exemplo: "Unpopular opinion: faculdade é perda de tempo"

TIPO 5: RELATABILITY (Identificação)
├── "Isso sou eu!"
├── Experiências universais
└── Vulnerabilidade autêntica
Exemplo: "POV: você procrastinando..."

TIPO 6: NEWS/TREND (Notícia/Tendência)
├── Primeiro a reportar algo
├── Opinião sobre evento atual
└── Participação em trend viral
Exemplo: Comentário sobre evento do momento
```

### Anatomia de um Post Viral

```
POST VIRAL COMPLETO

┌─────────────────────────────────────┐
│  HOOK (0-3 segundos)                │
│  ├── Pattern interrupt              │
│  ├── Promessa implícita             │
│  └── Gancho emocional               │
├─────────────────────────────────────┤
│  SETUP (3-10 segundos)              │
│  ├── Contexto mínimo necessário     │
│  ├── Aumenta curiosidade            │
│  └── Estabelece stakes              │
├─────────────────────────────────────┤
│  PAYOFF (10-45 segundos)            │
│  ├── Entrega da promessa            │
│  ├── Valor concreto                 │
│  └── Mantém ritmo                   │
├─────────────────────────────────────┤
│  TWIST/CLIMAX (se aplicável)        │
│  ├── Surpresa                       │
│  ├── Plot twist                     │
│  └── Revelação inesperada           │
├─────────────────────────────────────┤
│  CTA (últimos 3 segundos)           │
│  ├── Ação específica                │
│  ├── Razão para fazer               │
│  └── Urgência/motivação             │
└─────────────────────────────────────┘
```

### Loop de Retenção

```
RETENÇÃO = f(Curiosidade, Valor, Antecipação)

Técnicas para manter retenção alta:

1. OPEN LOOPS
   Abrir questão → Responder parcialmente → Abrir nova questão
   "Mas antes de explicar isso, preciso te contar..."

2. PATTERN VARIATION
   Mudar ritmo, tom ou visual a cada 3-7 segundos
   Evita que cérebro "desligue" por previsibilidade

3. TENSION BUILDING
   Criar e manter tensão narrativa
   "E então aconteceu algo inesperado..."

4. VALUE STACKING
   Cada segmento entrega valor independente
   Mesmo quem sai cedo, leva algo

5. CLIFFHANGERS
   Terminar antes da resolução completa
   "E o resultado? Comenta que eu conto na parte 2"
```

## Estratégias de Crescimento Orgânico

### Os 4 Pilares do Crescimento

```
CRESCIMENTO SUSTENTÁVEL

     CONTEÚDO           CONSISTÊNCIA
         │                   │
         ▼                   ▼
    ┌─────────────────────────────┐
    │                             │
    │      CRESCIMENTO            │
    │       ORGÂNICO              │
    │                             │
    └─────────────────────────────┘
         ▲                   ▲
         │                   │
    COMUNIDADE          CONVERSÃO

1. CONTEÚDO: Qualidade e relevância
2. CONSISTÊNCIA: Frequência e horário
3. COMUNIDADE: Engajamento genuíno
4. CONVERSÃO: De visitante para seguidor
```

### Estratégia de Conteúdo em Camadas

```
MODELO DE 3 CAMADAS

CAMADA 1 - ALCANCE (80% do conteúdo)
├── Reels, TikToks, posts virais
├── Objetivo: Novos olhos
├── Métricas: Alcance, impressões
└── Foco: Topo de funil

CAMADA 2 - RELACIONAMENTO (15% do conteúdo)
├── Carrosséis educativos, stories pessoais
├── Objetivo: Construir confiança
├── Métricas: Salvamentos, comentários
└── Foco: Meio de funil

CAMADA 3 - CONVERSÃO (5% do conteúdo)
├── CTAs diretos, ofertas, lançamentos
├── Objetivo: Ação específica
├── Métricas: Cliques, vendas
└── Foco: Fundo de funil
```

### O Método PASTOR para Crescimento

```
P - PROBLEMA
Identificar a dor principal da audiência
"Você está frustrado com..."

A - AMPLIFICAÇÃO
Amplificar as consequências do problema
"Isso está te custando..."

S - SOLUÇÃO
Apresentar sua abordagem única
"Existe uma forma diferente..."

T - TRANSFORMAÇÃO
Mostrar o resultado possível
"Imagine se você pudesse..."

O - OFERTA
O que você entrega (conteúdo)
"Por isso eu criei..."

R - RESPOSTA
CTA claro
"Siga para mais..."
```

### Estratégias por Fase de Crescimento

```
FASE 1: 0 - 1.000 seguidores
├── Foco em NICHAR fortemente
├── Postar 2-3x por dia
├── Comentar em 50+ posts por dia
├── Responder TODO comentário
├── Colaborar com criadores do mesmo tamanho
└── Objetivo: Encontrar seu conteúdo que funciona

FASE 2: 1.000 - 10.000 seguidores
├── Dobrar no que funciona
├── Introduzir séries de conteúdo
├── Criar hashtag proprietária
├── Iniciar email list
├── Fazer lives semanais
└── Objetivo: Construir comunidade core

FASE 3: 10.000 - 100.000 seguidores
├── Diversificar formatos
├── Colaborar com criadores maiores
├── Monetização inicial
├── Automatizar o que for possível
├── Treinar equipe (se aplicável)
└── Objetivo: Escalar sem perder autenticidade

FASE 4: 100.000+ seguidores
├── Posicionamento como autoridade
├── Criar produtos/serviços
├── Parcerias estratégicas
├── Conteúdo evergreen
├── Delegar criação tática
└── Objetivo: Construir marca pessoal/empresa
```

### O Framework "1000 True Fans"

```
CONCEITO (Kevin Kelly):
"Você só precisa de 1000 fãs verdadeiros para ter sucesso"

APLICAÇÃO EM REDES SOCIAIS:

1. Identifique seu "Superfã ideal"
   - Quem é?
   - O que quer?
   - Como você muda a vida dele?

2. Crie conteúdo para ELES (não para todos)
   - Profundidade > Amplitude
   - Específico > Genérico
   - Valor real > Engajamento vazio

3. Construa relacionamento direto
   - Responda DMs pessoalmente
   - Reconheça seguidores frequentes
   - Crie experiências exclusivas

4. Monetize através de valor genuíno
   - Produtos que resolvem problemas reais
   - Preço justo pelo valor entregue
   - Entrega excepcional
```

## Especialidades por Plataforma

### Instagram

#### Como o Algoritmo do Instagram Funciona (2024-2025)

```
SINAIS DO ALGORITMO INSTAGRAM

FEED E STORIES:
1. Relacionamento
   - Frequência de interações
   - DMs trocadas
   - Tags e menções mútuas
   - Tempo visualizando conteúdo do criador

2. Interesse
   - Histórico de engajamento com tipo de conteúdo
   - Tempo gasto em posts similares
   - Buscas relacionadas

3. Recência
   - Posts mais recentes têm prioridade
   - Mas conteúdo evergreen pode ressurgir

REELS:
1. Watch Time (mais importante)
   - % do vídeo assistido
   - Rewatches (assistir de novo)
   - Compartilhamentos para DM

2. Engajamento
   - Likes, comentários, salvamentos
   - Compartilhamentos em stories

3. Audio Trends
   - Usar áudios em alta aumenta distribuição
   - Original sounds podem viralizar

4. Valor de entretenimento
   - IA detecta qualidade visual
   - Considera edição e criatividade
```

**Formatos:**
| Formato | Dimensões | Ideal Para |
|---------|-----------|------------|
| Feed Quadrado | 1080x1080px | Citações, dicas |
| Feed Retrato | 1080x1350px | Carrosséis, fotos |
| Stories | 1080x1920px | Bastidores, enquetes |
| Reels | 1080x1920px | Tutoriais, trends |

**Estrutura de Carrossel:**
```
SLIDE 1 (CAPA)
├── Headline impactante
├── Promessa clara
└── Design que para o scroll

SLIDES 2-8 (CONTEÚDO)
├── 1 ideia por slide
├── Fonte 24pt+ (legível)
└── Progressão lógica

SLIDE 9 (RESUMO)
└── Recap dos pontos

SLIDE 10 (CTA)
├── Ação específica
└── Salvar/Compartilhar/Seguir
```

**Estrutura de Reels:**
```
0-1s: HOOK (visual ou sonoro)
1-3s: Setup do tema
3-12s: Conteúdo principal
12-15s: Payoff + CTA
```

**Hashtags Instagram:**
- Mix: 3 grandes + 5 médias + 7 pequenas
- Total: 10-15 hashtags
- Específicas > Genéricas

### LinkedIn

#### Como o Algoritmo do LinkedIn Funciona (2024-2025)

```
SINAIS DO ALGORITMO LINKEDIN

1. DWELL TIME (Tempo de Permanência)
   - Quanto tempo gastam lendo seu post
   - Posts longos com valor performam bem
   - Texto > Link externo (mantém na plataforma)

2. ENGAJAMENTO INICIAL (Golden Hour)
   - Primeira hora é CRÍTICA
   - Comentários > Likes
   - Comentários longos > Comentários curtos

3. RELEVÂNCIA
   - Conexões 1º grau veem primeiro
   - Depois se espalha para 2º e 3º grau
   - Nicho específico performa melhor

4. FORMATO PREFERIDO (ordem de prioridade)
   - Text posts com formatação
   - Documentos (carrosséis PDF)
   - Imagens nativas
   - Vídeos nativos
   - Links externos (pior performance)

DICA AVANÇADA:
LinkedIn penaliza links externos no post.
Estratégia: Postar conteúdo, link no primeiro comentário.
```

**Estrutura de Post:**
```
[HOOK] Primeira linha forte
↓ (quebra de linha para suspense)

[CONTEXTO] Situação ou problema

[INSIGHT] Sua perspectiva única

[DESENVOLVIMENTO] 3-5 pontos ou história

[CONCLUSÃO] Aprendizado/takeaway

[CTA] Pergunta ou chamada para ação

#hashtag1 #hashtag2 #hashtag3 (máx 5)
```

**O que performa:**
- Histórias pessoais (vulnerabilidade)
- Lições aprendidas
- Opiniões contrárias (controversas)
- Dados e insights originais
- Celebrações (com humildade)

#### Táticas Avançadas LinkedIn

```
FORMATOS QUE VIRALIZAM:

1. POST DE LISTA
   "10 lições que aprendi em 10 anos como [profissão]"
   ├── Uma lição por linha
   ├── Começar cada linha com emoji ou número
   └── Fechar com reflexão pessoal

2. POST CONTROVERSO (calculated controversy)
   "Unpopular opinion: [afirmação que desafia o status quo]"
   ├── Afirmação forte na primeira linha
   ├── Argumentação sólida
   └── Convite ao debate

3. POST DE HISTÓRIA
   "Fui demitido na sexta-feira passada."
   ├── Gancho emocional
   ├── Jornada com conflito
   └── Lição universal

4. POST DE FRAMEWORK
   "O método [NOME] que uso para [resultado]"
   ├── Sigla memorável
   ├── Cada letra = um passo
   └── Exemplo prático

5. POST DE DOCUMENTO (carrossel)
   "Salvei este post porque ninguém fala sobre [tema]"
   ├── 10-15 slides
   ├── Uma ideia por slide
   └── CTA para salvar/seguir
```

FORMATOS QUE VIRALIZAM:

4. LISTICLE

   (Maioria gratuita)"
   ├── Uma ferramenta por tweet

5. QUOTE TWEET STRATEGY

### TikTok

#### Como o Algoritmo do TikTok Funciona (2024-2025)

```
SINAIS DO ALGORITMO TIKTOK (For You Page)

1. WATCH TIME (Mais importante)
   - % do vídeo assistido

AMPLIFICATION RATE

* Quanto seu conteúdo é amplificado

CONVERSION RATE

* Medir efetividade de CTAs

- Nomes de boards com keywords (ex: "Receitas Fit Fáceis" ao invés de "Comida")
- Descrição de board com keywords (até 500 caracteres)
- Board covers consistentes com identidade visual
- Boards secretos para curar conteúdo antes de publicar
- Organizar pins em seções dentro dos boards
- Manter boards ativos (pinar regularmente em cada um)

### Pinterest SEO

**Otimização para busca no Pinterest:**
- Pinterest funciona como um motor de busca visual
- Title do pin: keyword no início (até 100 caracteres)
- Descrição: até 500 caracteres com keywords naturais
- Alt text da imagem: descritivo com keywords
- Nome do arquivo da imagem: descritivo (ex: receita-bolo-chocolate.jpg)
- Boards otimizados com keywords no nome e descrição
- Consistência de publicação > volume

**Keywords no Pinterest:**
- Usar barra de busca do Pinterest para sugestões
- Pinterest Trends para identificar tendências sazonais
- Palavras-chave long-tail performam bem
- Pesquisar o que concorrentes usam

### Idea Pins

**O que são:** Formato multi-página (até 20 páginas), similar a Stories mas permanente.

**Boas práticas:**
- Usar para tutoriais passo a passo
- Conteúdo educativo e inspiracional
- Incluir texto, vídeo e imagens
- Tagging de produtos (se aplicável)
- CTA para seguir o perfil
- Não permite links externos (foco em engajamento)

**Estrutura:**
```
PÁGINA 1: Capa com hook visual
PÁGINAS 2-8: Conteúdo passo a passo
PÁGINA 9: Resultado final ou resumo
PÁGINA 10: CTA (seguir, salvar)
```

## Gestão de Crise nas Redes

### Framework HEAR

```
H - HEAR (Ouvir)
├── Monitorar menções e comentários
├── Identificar o problema rapidamente
├── Entender a gravidade da situação
└── Coletar todas as informações antes de agir

E - EMPATHIZE (Empatizar)
├── Reconhecer o sentimento do público
├── Validar a frustração ou preocupação
├── Não minimizar o problema
└── Mostrar que a marca se importa

A - ACT (Agir)
├── Tomar ação concreta para resolver
├── Comunicar as medidas sendo tomadas
├── Definir prazos realistas
└── Envolver equipe responsável

R - RESPOND (Responder)
├── Resposta pública transparente
├── Atualizar conforme evolução
├── Agradecer pelo feedback
└── Documentar para prevenção futura
```

### Templates de Resposta a Comentários Negativos

**Reclamação sobre produto/serviço:**
```
"Olá, [Nome]. Lamentamos muito pela sua experiência. Isso não
reflete o padrão que buscamos. Vamos resolver isso para você.
Por favor, envie uma DM com seus dados para que nossa equipe
possa atender seu caso com prioridade."
```

**Crítica pública:**
```
"[Nome], agradecemos seu feedback. Levamos todas as opiniões a
sério e queremos entender melhor o que aconteceu. Podemos
conversar por mensagem direta para resolver essa situação?"
```

**Comentário ofensivo:**
```
"Respeitamos todas as opiniões, mas pedimos que a comunicação
seja feita de forma respeitosa. Estamos disponíveis para
conversar de forma construtiva sobre qualquer questão."
```

**Fake news sobre a marca:**
```
"Gostaríamos de esclarecer que essa informação não procede.
[Inserir fatos e dados reais]. Estamos à disposição para
esclarecer qualquer dúvida. Transparência é um dos nossos
valores."
```

### Quando Responder vs Quando Ignorar

| Situação | Ação | Motivo |
|----------|------|--------|
| Reclamação legítima | Responder imediatamente | Resolver e mostrar comprometimento |
| Pergunta genuína | Responder | Oportunidade de engajamento |
| Crítica construtiva | Responder e agradecer | Mostra maturidade da marca |
| Troll óbvio | Ignorar | Responder alimenta o troll |
| Spam | Deletar/denunciar | Poluição do perfil |
| Comentário ofensivo | Ocultar + resposta padrão | Manter ambiente seguro |
| Crise viral | Resposta oficial urgente | Controlar narrativa |
| Elogio | Responder e agradecer | Fortalecer relacionamento |

### Processo de Escalação

```
NÍVEL 1 - Community Manager
├── Comentários rotineiros
├── Dúvidas simples
└── Elogios e interações

NÍVEL 2 - Coordenação de Social
├── Reclamações recorrentes
├── Menções de influenciadores
└── Potencial de viralização negativa

NÍVEL 3 - Gerência/Diretoria
├── Crise instalada
├── Repercussão na imprensa
├── Ameaças legais
└── Risco à reputação da marca

NÍVEL 4 - Assessoria Jurídica + RP
├── Crise de grande proporção
├── Processos judiciais
└── Comunicados oficiais
```

### Plano de Recuperação Pós-Crise

1. **Análise interna** - Documentar o que aconteceu e por quê
2. **Comunicado de resolução** - Informar o público que o problema foi resolvido
3. **Mudanças implementadas** - Mostrar ações concretas para evitar recorrência
4. **Monitoramento intensivo** - Acompanhar menções por 30 dias
5. **Conteúdo positivo** - Retomar agenda de conteúdo gradualmente
6. **Relatório de crise** - Documentar aprendizados para o time
7. **Atualizar protocolo** - Revisar processo de crise com base nos aprendizados

## Community Management

### Estratégias de Engajamento

**Proativas:**
- Fazer perguntas nos posts (abertas, não sim/não)
- Criar enquetes e votações regulares
- Responder stories dos seguidores
- Comentar em perfis de seguidores ativos
- Criar séries de conteúdo (ex: "Dica de Terça")

**Reativas:**
- Responder todos os comentários em até 2 horas
- Curtir comentários dos seguidores
- Repostar conteúdo dos seguidores (UGC)
- Responder DMs em até 24 horas
- Agradecer compartilhamentos

### Templates de Resposta

**Dúvida sobre produto:**
```

### Automação de FAQ

- Política de troca
- Como comprar
- Contato de suporte

**Chatbot para DMs:**
```
FLUXO BÁSICO:
├── Saudação automática
├── Menu de opções
│   ├── 1. Comprar
│   ├── 2. Dúvidas
│   ├── 3. Suporte
│   └── 4. Falar com humano
├── Respostas para opções 1-3
└── Transição para atendente humano (opção 4)
```

**Dicas:**
- Automação para perguntas repetitivas
- Sempre oferecer opção de atendimento humano
- Personalizar mensagens (não parecer robótico)
- Revisar e atualizar respostas mensalmente

### Incentivos para Conteúdo Gerado pelo Usuário (UGC)

- Criar hashtag da marca para clientes usarem
- Repostar conteúdo de clientes (com permissão)
- Concursos de fotos/vídeos com o produto
- Programa de embaixadores da marca
- Desconto ou brinde para quem postar
- Destacar "cliente do mês" nos stories
- Criar template para clientes personalizarem
- Responder e agradecer todo UGC publicamente

### Diretrizes de Moderação

**Regras da comunidade:**
- Respeito mútuo obrigatório
- Proibido discurso de ódio, discriminação e assédio
- Sem spam ou autopromoção não autorizada
- Sem fake news ou informações enganosas
- Conteúdo adulto ou violento será removido

**Ações de moderação:**
| Infração | Primeira vez | Reincidência | Terceira vez |
|----------|-------------|--------------|--------------|
| Spam | Aviso + remoção | Silenciamento 24h | Ban |
| Linguagem ofensiva | Aviso | Silenciamento 48h | Ban |
| Discurso de ódio | Remoção + aviso | Ban | - |
| Assédio | Ban imediato | - | - |
| Autopromoção | Aviso | Remoção | Silenciamento |

**Boas práticas de moderação:**
- Documentar todas as ações tomadas
- Ser consistente na aplicação das regras
- Comunicar regras de forma clara e visível
- Revisar regras trimestralmente
- Treinar equipe de moderação regularmente
- Manter registro de usuários banidos
