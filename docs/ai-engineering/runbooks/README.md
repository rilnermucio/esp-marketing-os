# Runbooks

Procedimentos de diagnóstico pra quando algo quebra. Um runbook nasce de incidente real; o objetivo é a próxima pessoa (ou modelo) resolver em minutos o que custou horas na primeira vez.

| Runbook | Cobre |
|---|---|
| [install-failure-debug.md](install-failure-debug.md) | Install do plugin falhando (marketplace, validação, sync) |
