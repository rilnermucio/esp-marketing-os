# Gary Provost - Frameworks

## 1. A Escala Musical de Frases

O framework central de Provost para criar ritmo.

### As 5 Notas

| Tipo de Frase | Palavras | Efeito |
|---------------|----------|--------|
| **Ultra-curta** | 1-3 palavras | Impacto. Ponto final. |
| **Curta** | 4-7 palavras | Clareza direta e concisa. |
| **Média** | 8-15 palavras | Desenvolvimento de uma ideia simples com contexto. |
| **Longa** | 16-25 palavras | Envolvimento progressivo que sustenta atenção enquanto aprofunda. |
| **Expandida** | 26+ palavras | Narrativa completa que constrói, conecta e entrega a ideia em toda sua complexidade — usada estrategicamente para momentos de máximo impacto. |

### Regra de Composição

Nunca use mais de 2 frases do mesmo tipo em sequência.

**Padrão ideal para parágrafos de alto impacto:**
```
Curta.
Média que desenvolve.
Ultra-curta.
Longa que aprofunda e envolve completamente o leitor na ideia.
Curta para concluir.
```

## 2. Hierarquia de Força da Palavra

Da mais fraca para a mais forte:

```
Adjetivo genérico          (fraco)   → "muito bom"
Adjetivo específico        ↓         → "extraordinariamente preciso"
Substantivo forte          ↓         → "maestria"
Verbo fraco                ↓         → "ele andou rapidamente"
Verbo forte                ↓         → "ele correu"
Verbo preciso + cena       (forte)   → "ele atravessou a sala em 3 passos, derrubou a cadeira"
```

### Exercício de Substituição

Para cada adjetivo no seu texto, pergunte:
1. Posso substituir por um verbo?
2. Posso mostrar em vez de dizer?
3. Posso dar um exemplo concreto?

**Antes:** "O produto é muito eficiente."
**Depois:** "Em 8 minutos, o produto fez o que levava 2 horas antes."

## 3. Framework Mostrar vs Contar

### Quando CONTAR (raro)
- Informação factual objetiva
- Transições rápidas
- Contexto necessário mas não central

### Quando MOSTRAR (sempre que possível)
- Emoções do personagem ou cliente
- Qualidade de um produto
- Credibilidade do autor
- Resultados e benefícios

### Fórmula de Transformação

```
CONTAR: [sujeito] + [adjetivo/estado emocional]
   ↓
MOSTRAR: [sujeito] + [ação física/específica que revela o estado]
```

**Exemplos:**

| Contar | Mostrar |
|--------|---------|
| "O cliente ficou satisfeito" | "O cliente voltou na semana seguinte e trouxe o irmão" |
| "É fácil de usar" | "Minha mãe de 70 anos configurou sozinha em 4 minutos" |
| "O produto é durável" | "Caiu 3 vezes no chão de concreto. Ainda funciona." |
| "Senti medo" | "Minhas mãos suaram. Não conseguia soltar o corrimão." |

## 4. Limpeza de Texto (Word Economy)

### Frases Parasitas — Eliminar Sempre

| Parasita | Versão limpa |
|----------|-------------|
| "É importante ressaltar que..." | Comece direto com a afirmação |
| "No que diz respeito a..." | Nomeie o assunto diretamente |
| "De certa forma podemos dizer que..." | Diga diretamente |
| "Com o objetivo de..." | Use "para" |
| "Na medida em que..." | Use "porque" ou "quando" |
| "Em função disso..." | Use "por isso" |
| "Tendo em vista que..." | Use "como" ou "porque" |

### Regra dos Três Passes

1. **Primeiro passe:** Escreva tudo, sem filtro
2. **Segundo passe:** Corte 20% das palavras
3. **Terceiro passe:** Substitua 3 adjetivos por verbos

## 5. Estrutura de Parágrafo

### Padrão Ideal

```
FRASE TÓPICO (o que este parágrafo diz)
   ↓
DESENVOLVIMENTO (prova, exemplo, detalhe)
   ↓
CONCLUSÃO ou GANCHO (encerra ou puxa para o próximo parágrafo)
```

### Comprimento

- **Mínimo:** 2 frases (exceto para efeito dramático)
- **Máximo:** 5 frases
- **Ideal:** 3-4 frases com variação de comprimento

## Resumo para Aplicação

| Framework | Use quando... |
|-----------|---------------|
| Escala Musical | Revisar qualquer texto para melhorar ritmo |
| Hierarquia de Força | Substituir adjetivos fracos por verbos fortes |
| Mostrar vs Contar | Descrever benefícios, emoções ou qualidade |
| Limpeza de Texto | Revisar textos longos ou "pesados" |
| Estrutura de Parágrafo | Construir cada parágrafo com intenção |
