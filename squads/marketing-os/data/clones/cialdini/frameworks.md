# Robert Cialdini - Frameworks

## 1. Os 7 Princípios de Influência

O sistema central de Cialdini — base de tudo que ele ensina.

### O Mapa dos Princípios

| Princípio | Gatilho Central | Aplicação em Copy |
|-----------|----------------|-------------------|
| **Reciprocidade** | "Você me deu algo" | Ofereça valor antes de pedir |
| **Comprometimento & Consistência** | "Eu disse que faria isso" | Micro-compromissos que levam ao macro |
| **Prova Social** | "Outras pessoas estão fazendo" | Depoimentos, números, comportamento de pares |
| **Autoridade** | "Esse é um especialista" | Credenciais, publicações, experiência |
| **Afinidade** | "Eu gosto dessa pessoa" | Semelhança, elogios genuínos, familiaridade |
| **Escassez** | "Isso pode acabar" | Limitação real de tempo, quantidade ou acesso |
| **Unidade** | "Somos do mesmo grupo" | Identidade compartilhada, "nós" vs "eles" |

## 2. Framework de Reciprocidade Estratégica

Cialdini demonstrou que dar primeiro cria uma obrigação psicológica poderosa — mas apenas quando feito corretamente.

### As 3 Condições para Reciprocidade Eficaz

```
CONDIÇÃO 1 — INESPERADO
O presente deve surpreender, não ser esperado.
→ "Aqui está algo que preparei para você" (sem aviso prévio)
→ Evite: brindes anunciados com antecedência perdem o poder

CONDIÇÃO 2 — PERSONALIZADO
Deve parecer feito especificamente para o receptor.
→ "Vi que você [situação específica] e pensei nisto"
→ Evite: presentes genéricos diminuem o efeito

CONDIÇÃO 3 — SIGNIFICATIVO
Deve ter valor real — não ser marketing disfarçado de presente.
→ Conteúdo que resolve um problema real, não um teaser
→ Evite: "brindes" que são apenas argumentos de venda embalados
```

### Sequência de Reciprocidade em Copy

```
PASSO 1: ENTREGUE VALOR PRIMEIRO
Conteúdo real, ferramenta útil, insight genuíno — sem pedir nada

PASSO 2: ESTABELEÇA O QUE FOI DADO
"Você acabou de [ação] / aprendeu [conceito] / descobriu [insight]"

PASSO 3: FAÇA A TRANSIÇÃO NATURAL
"E o que você viu até agora é apenas a superfície do que [produto] faz"

PASSO 4: APRESENTE A OFERTA
Agora o leitor está psicologicamente inclinado a retribuir com atenção
```

## 3. Framework de Comprometimento Progressivo

A escada de micro-sim que leva ao macro-sim.

### A Lógica do Princípio

Pessoas agem de forma consistente com o que já disseram ou fizeram. Uma vez que concordaram com algo pequeno, têm dificuldade psicológica de recuar.

### Escada de Comprometimento

```
DEGRAU 1: MICRO-COMPROMISSO COGNITIVO
"Você concorda que [afirmação óbvia]?"
Exemplo: "Você concorda que resultado X é importante para quem faz Y?"

DEGRAU 2: MICRO-COMPROMISSO COMPORTAMENTAL
Clique, preenchimento de formulário, resposta simples
Exemplo: "Diga nos comentários: qual é seu maior desafio com X?"

DEGRAU 3: COMPROMISSO DE IDENTIDADE
"Pessoas como você [identidade] normalmente [comportamento]"
Exemplo: "Profissionais que levam X a sério costumam Y"

DEGRAU 4: OFERTA CONSISTENTE COM OS COMPROMISSOS ANTERIORES
Apresenta o produto como a continuação lógica do que a pessoa já disse que quer
```

### Aplicação em Email Sequence

```
Email 1: Pergunta + conteúdo (micro-compromisso: abrir e ler)
Email 2: "Como você responderia X?" (compromisso comportamental)
Email 3: "Para quem [identidade que o leitor quer ter]..." (compromisso de identidade)
Email 4: Oferta framed como "a próxima passo natural para quem já fez X"
```

## 4. Framework de Prova Social Estratificada

Não toda prova social tem o mesmo peso. Cialdini identificou que a prova social é mais poderosa quando vem de pessoas similares ao receptor.

### Os 4 Níveis de Prova Social

```
NÍVEL 1 — NÚMEROS (mais fraco, mas amplo)
"Mais de 10.000 clientes"
→ Use para: escala, legitimidade geral

NÍVEL 2 — DEPOIMENTO DE SEMELHANTE
"[Nome], [profissão/situação similar ao leitor], diz que..."
→ Use para: identificação — "se funcionou para alguém como eu..."

NÍVEL 3 — CASO ESPECÍFICO COM RESULTADO MENSURÁVEL
"João, gerente de médias empresas, aumentou em 34% em 3 meses"
→ Use para: credibilidade alta + especificidade

NÍVEL 4 — PROVA SOCIAL DE AUTORIDADE
Especialista reconhecido ou instituição valida o produto
→ Use para: legitimidade em mercados céticos
```

### Framework de Apresentação de Prova Social

```
PERFIL: Quem é essa pessoa (similar ao leitor ideal)
SITUAÇÃO ANTES: Onde estava antes do produto
AÇÃO: O que fez com o produto
RESULTADO ESPECÍFICO: Número, prazo, transformação mensurável
RECOMENDAÇÃO: Por que indica (em linguagem própria)
```

## 5. Framework de Escassez Ética

Cialdini é rigoroso: escassez falsa não apenas é antiética — ela destrói confiança a longo prazo.

### Escassez Real vs. Fabricada

| Tipo | Exemplo | Efeito a Longo Prazo |
|------|---------|---------------------|
| **Escassez real de quantidade** | "Últimas 47 unidades em estoque" | Positivo — confiança mantida |
| **Escassez real de tempo** | "Bônus disponível até sexta-feira 23h59" | Positivo — se o prazo for respeitado |
| **Escassez de acesso** | "Turma fechada — próxima abertura em março" | Positivo — se for verdade |
| **Escassez fabricada** | "Oferta expira em 10 minutos" (que se renova) | Destrutivo — erode credibilidade |

### Como Comunicar Escassez Real

```
1. ESPECIFIQUE O MECANISMO DA ESCASSEZ
"Não reabremos esta turma porque [razão real]"

2. SEJA ESPECÍFICO NO NÚMERO OU DATA
"14 vagas" não "vagas limitadas"
"Até 31 de março" não "em breve"

3. EXPLIQUE POR QUE IMPORTA AGIR AGORA
Não apenas "vai acabar" — mas "o custo de esperar é [consequência específica]"

4. CUMPRA O QUE PROMETEU
Se disse que fecha sexta, fecha sexta. A credibilidade futura depende disso.
```

## 6. Framework de Unidade (7° Princípio)

O mais poderoso e menos explorado dos princípios de Cialdini, introduzido em *Pre-Suasion*.

### A Diferença entre Afinidade e Unidade

- **Afinidade:** "Gosto de você porque somos parecidos"
- **Unidade:** "Somos do mesmo grupo — o que me acontece, acontece a nós"

Unidade é identidade compartilhada. É mais profunda que semelhança.

### Gatilhos de Unidade em Copy

```
IDENTIDADE DE GRUPO
"Para quem escolheu [estilo de vida/profissão/missão]..."
→ Não apenas "cliente" — mas membro de uma tribo

CO-CRIAÇÃO
"Este produto foi desenvolvido com [número] de pessoas como você"
→ O comprador se torna parte da origem, não apenas destinatário

MISSÃO COMPARTILHADA
"Estamos construindo [objetivo maior] juntos"
→ Comprar é participar de algo além da transação

HERANÇA FAMILIAR OU CULTURAL
"Para quem cresceu acreditando que [valor]..."
→ Aciona pertencimento profundo
```

### Sequência de Unidade

```
PASSO 1: NOMEIE O GRUPO
"Somos [identidade]" — não "nossos clientes"

PASSO 2: DEFINA A MISSÃO OU VALOR COMPARTILHADO
O que une o grupo além do produto

PASSO 3: POSICIONE O PRODUTO COMO EXTENSÃO DA IDENTIDADE
Comprar = agir como membro do grupo

PASSO 4: TRATE O LEITOR COMO MEMBRO, NÃO COMO PROSPECT
Linguagem de "nós" e "juntos" em vez de "você vai receber"
```

## Resumo para Aplicação

| Framework | Use quando... |
|-----------|---------------|
| 7 Princípios (visão geral) | Auditoria de copy existente |
| Reciprocidade Estratégica | Sequência de nutrição, lead magnets, conteúdo gratuito |
| Comprometimento Progressivo | Funis de email, sequências longas, webinars |
| Prova Social Estratificada | Páginas de vendas, depoimentos, studies de caso |
| Escassez Ética | Qualquer oferta com prazo ou quantidade limitada |
| Unidade | Comunicação de marca, posicionamento de comunidade |
