# Donald Miller - Frameworks

## 1. StoryBrand 7-Part Framework (SB7)

O framework central de Miller. Transforma qualquer mensagem de negócio em uma história clara usando 7 elementos universais de storytelling.

```
1. Um PERSONAGEM (cliente)
2. Tem um PROBLEMA
3. E encontra um GUIA (sua marca)
4. Que lhe dá um PLANO
5. E o CHAMA PARA AÇÃO
6. Que resulta em SUCESSO
7. E o ajuda a evitar FRACASSO
```

### Componentes Detalhados

| Componente | Descrição | Pergunta-Chave |
|------------|-----------|----------------|
| **Personagem** | O cliente como herói da história | O que o cliente quer? |
| **Problema** | O vilão e os 3 níveis de problema | Qual é o problema externo, interno e filosófico? |
| **Guia** | A marca com empatia e autoridade | Como demonstramos que entendemos E que somos competentes? |
| **Plano** | 3 passos claros para o cliente seguir | Quais são os 3 passos simples? |
| **Ação** | CTA direto e transicional | O que pedimos ao cliente para fazer? |
| **Sucesso** | A visão de vida transformada | Como a vida do cliente melhora? |
| **Fracasso** | O que está em jogo se não agir | O que o cliente perde se não agir? |

### Os 3 Níveis de Problema

Todo problema tem 3 camadas que devem ser endereçadas:

| Nível | Descrição | Exemplo (consultor financeiro) |
|-------|-----------|-------------------------------|
| **Externo** | O problema prático e tangível | "Não tenho controle sobre minhas finanças" |
| **Interno** | O sentimento causado pelo problema | "Sinto vergonha de não saber gerenciar meu dinheiro" |
| **Filosófico** | A injustiça ou questão maior | "Trabalho duro demais para viver com essa insegurança" |

### Aplicação na Copy

Para cada peça de comunicação, preencha o BrandScript:

```
Meu cliente quer [DESEJO].
O problema é que [VILÃO/OBSTÁCULO].
Isso faz ele sentir [FRUSTRAÇÃO INTERNA].
E não é justo porque [PROBLEMA FILOSÓFICO].
Nós entendemos como é [EMPATIA].
Já ajudamos [NÚMERO] de [CLIENTES] a [RESULTADO] [AUTORIDADE].
O plano é simples: 1. [PASSO 1] 2. [PASSO 2] 3. [PASSO 3].
[CTA DIRETO].
Imagine [VISÃO DE SUCESSO].
Não deixe [RISCO DE FRACASSO].
```

## 2. One-Liner

Uma frase que comunica exatamente o que sua empresa faz em 3 partes.

### Estrutura

```
Problema + Solução + Resultado
```

### Fórmula

```
"A maioria dos [clientes] luta com [problema].
Nós oferecemos [solução].
Para que eles possam [resultado desejado]."
```

### Exemplos

- "A maioria dos donos de negócios perde vendas por mensagens confusas. Nós clarificamos sua mensagem com o framework StoryBrand. Para que mais clientes entendam por que devem escolher você."
- "Empreendedores gastam fortunas em marketing que não funciona. Criamos landing pages baseadas em storytelling. Para que cada real investido gere vendas reais."

### Regras

1. Deve caber em 2-3 frases
2. Comece com o problema (gera identificação)
3. A solução deve ser simples de entender
4. O resultado deve ser aspiracional mas realista

## 3. Website Wireframe (5 Seções)

Framework para estruturar qualquer landing page ou site.

### Estrutura

```
SEÇÃO 1: Header
- Headline: O que você oferece
- Subheadline: Como melhora a vida do cliente
- CTA principal
- Imagem aspiracional

SEÇÃO 2: O Problema
- Identifique a dor do cliente
- Nomeie o vilão
- Mostre empatia

SEÇÃO 3: O Plano
- 3 passos simples
- Visualmente claro
- Remova a complexidade

SEÇÃO 4: Prova
- Depoimentos
- Logos de clientes
- Números e resultados

SEÇÃO 5: CTA Final
- Repita o CTA
- Adicione parágrafo de fechamento
- Mostre o que está em jogo
```

## 4. Email Sequence (Nurture Campaign)

Framework para sequência de emails que constrói confiança e converte.

### Estrutura de 6 Emails

```
Email 1: Entregue valor (lead magnet)
Email 2: Identifique o problema
Email 3: Posicione-se como guia (história de sucesso)
Email 4: Apresente o plano
Email 5: Mostre o que está em jogo (sucesso vs fracasso)
Email 6: CTA direto com urgência
```

### Regra de Ouro

> "Cada email deve ter apenas um objetivo. Uma ideia. Um CTA."

## 5. Marketing Made Simple Framework

Sistema completo de funil de vendas em 3 etapas.

### As 3 Etapas

| Etapa | Objetivo | Ferramenta |
|-------|----------|------------|
| **Curiosidade** | Criar interesse | One-liner, ads, headlines |
| **Esclarecimento** | Explicar a oferta | Landing page, wireframe, vídeo |
| **Compromisso** | Converter em cliente | Email sequence, CTA, oferta |

### Sales Funnel Simplificado

```
1. Lead Generator (PDF, vídeo, checklist gratuito)
   ↓
2. Nurture Email Sequence (5-6 emails educativos)
   ↓
3. Sales Email Sequence (3-4 emails de conversão)
   ↓
4. Página de vendas (wireframe StoryBrand)
```
