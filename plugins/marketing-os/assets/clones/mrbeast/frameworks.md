# MrBeast (Jimmy Donaldson) - Frameworks

## 1. Retention Engineering (Engenharia de Retenção)

O framework central de MrBeast. Cada segundo de um vídeo é projetado para manter o espectador assistindo. Se a retenção cair, o vídeo fracassa, não importa quantas impressões tenha.

```
Retenção = f(Gancho, Escalada, Ganchos de Curiosidade, Payoff)
```

### Os 4 Pilares da Retenção

| Pilar | Descrição | Onde no Vídeo |
|-------|-----------|---------------|
| **Gancho** | Premissa clara + aposta alta em 5-30 segundos | Primeiros 30 segundos |
| **Escalada** | Tensão crescente, cada momento maior que o anterior | Ao longo de todo o vídeo |
| **Ganchos de Curiosidade** | Frases que criam expectativa sobre o que vem a seguir | A cada 2-3 minutos |
| **Payoff** | Entrega emocional que justifica ter assistido até o final | Final do vídeo |

### Curva de Retenção Ideal

```
100% ┤ ████
     │    ██
 80% ┤      ███
     │         ██
 60% ┤           ████
     │               ███
 40% ┤                  ███████
     │
 20% ┤
     │
  0% ┤───────────────────────────→
     0s  30s  2min  5min  10min  final

Meta: nunca cair abaixo de 40% em qualquer ponto do vídeo
Ideal: manter acima de 50% até o final
```

### Regras de Retenção

1. **Regra dos 30 Segundos** - Se a retenção cai mais de 20% nos primeiros 30 segundos, o gancho falhou
2. **Regra do Vale** - Se houver um vale na curva, identifique o que causou e elimine em vídeos futuros
3. **Regra do Re-hook** - A cada 2-3 minutos, plante um gancho de curiosidade para evitar quedas
4. **Regra do Payoff** - O final deve ter um pico de emoção que supere a média do vídeo

## 2. Thumbnail + Title Framework (Click-Through Rate)

O CTR (Click-Through Rate) determina quantas pessoas clicam quando veem o vídeo. Sem clique, não existe view. Sem view, não existe retenção.

### Regras de Thumbnail

```
THUMBNAIL EFICAZ = Emoção Clara + Contraste Visual + Curiosidade + Simplicidade
```

| Elemento | Regra | Exemplo |
|----------|-------|---------|
| **Emoção** | Um rosto deve transmitir a emoção principal do vídeo | Boca aberta de choque, olhos arregalados |
| **Contraste** | Cores contrastantes que se destacam no feed | Fundo azul com elemento vermelho |
| **Texto** | Máximo 3-4 palavras grandes e legíveis | "R$1 MILHÃO" |
| **Composição** | Elemento central claro, sem poluição visual | Uma imagem focal, espaço limpo |
| **Curiosidade** | Deve gerar uma pergunta na mente do espectador | Mostrar a situação, não o resultado |

### Regras de Título

```
TÍTULO EFICAZ = Número/Superlativo + Conceito Simples + Curiosity Gap
```

| Princípio | Bom Exemplo | Mau Exemplo |
|-----------|-------------|-------------|
| **Curto** | "Sobrevivi 50 Horas no Oceano" | "Minha Experiência Incrível Sobrevivendo 50 Horas Perdido no Oceano" |
| **Número grande** | "Dei R$1 Milhão Para Estranhos" | "Dei Dinheiro Para Algumas Pessoas" |
| **Curiosidade** | "Abri o Cofre e Encontrei..." | "Abri o Cofre e Encontrei R$100K" |
| **Complementa thumb** | Thumbnail mostra choque + Título explica o porquê | Título repete exatamente o que está na thumbnail |

### Teste A/B de Thumbnail

```
1. Crie 3-5 versões de thumbnail
2. Teste com amostra de público (poll, comunidade)
3. Analise qual gera mais curiosidade
4. Após publicar, monitore CTR nas primeiras 2 horas
5. Troque a thumbnail se CTR < 5%
```

## 3. Video Structure Framework (Estrutura de Vídeo)

### Estrutura para Vídeo Longo (10-25 minutos)

```
[0:00-0:30] GANCHO
  → Premissa em 1 frase
  → Aposta clara (o que está em jogo)
  → Visual impactante

[0:30-2:00] SETUP
  → Contexto rápido
  → Apresentar participantes/situação
  → Regras do desafio

[2:00-5:00] DESENVOLVIMENTO INICIAL
  → Primeiras ações/rodadas
  → Primeiro gancho de curiosidade
  → Primeira surpresa ou twist

[5:00-15:00] ESCALADA
  → Cada segmento mais intenso que o anterior
  → Eliminações, revelações, mudanças de regra
  → Ganchos de curiosidade a cada 2-3 minutos
  → Momentos emocionais intercalados

[15:00-20:00] CLÍMAX
  → Tensão máxima
  → Poucos participantes/alto risco
  → O momento que tudo pode mudar

[20:00-25:00] PAYOFF + ENCERRAMENTO
  → Resolução emocional
  → Reação genuína
  → Mensagem final breve
  → Não arrastar após o payoff
```

### Estrutura para Short/Reels (15-60 segundos)

```
[0:00-0:03] GANCHO VISUAL
  → Imagem ou ação que prende imediatamente
  → Sem introdução verbal

[0:03-0:10] PREMISSA
  → O que está acontecendo em 1 frase
  → A aposta

[0:10-0:40] EXECUÇÃO
  → A ação acontecendo
  → Mini-escalada

[0:40-0:60] TWIST + PAYOFF
  → Surpresa ou resultado
  → Reação emocional
  → Potencial de loop (o final conecta ao início)
```

## 4. Content Ideation Framework

Sistema para gerar ideias de vídeo com potencial viral.

### A Fórmula de Ideação

```
IDEIA VIRAL = Conceito Familiar × Escala Absurda + Fator Emocional
```

### Categorias de Conteúdo que Funcionam

| Categoria | Fórmula | Exemplo |
|-----------|---------|---------|
| **Desafio** | "Último a [ação] ganha [prêmio]" | "Último a sair do círculo ganha R$500K" |
| **Extremo** | "[Número absurdo] de [coisa]" | "Comprei o carro mais caro do mundo" |
| **Comparação** | "[Coisa barata] vs [coisa cara]" | "R$1 vs R$100.000 casa" |
| **Filantropia** | "Dei [algo incrível] para [pessoa]" | "Dei uma casa para quem precisava" |
| **Sobrevivência** | "Sobrevivi [tempo] em [lugar extremo]" | "Sobrevivi 7 dias numa ilha deserta" |
| **Recriação** | "[Referência pop] na vida real" | "Squid Game na vida real" |

### Filtro de Viabilidade

Antes de produzir, toda ideia deve passar por 5 perguntas:

```
1. A thumbnail é óbvia? (Se não consegue imaginar a thumb, descarte)
2. O título cabe em 8-12 palavras? (Se precisa de mais, simplifique)
3. O gancho é claro em 5 segundos? (Se demora para explicar, descarte)
4. Existe escalada natural? (Se é flat, descarte)
5. O payoff é emocional? (Se é só informação, repensar)
```

## 5. Re-hook System (Sistema de Re-engajamento)

Framework para plantar ganchos de curiosidade ao longo do vídeo e impedir quedas de retenção.

### Tipos de Re-hooks

| Tipo | Exemplo | Quando Usar |
|------|---------|-------------|
| **Preview** | "E vocês não vão acreditar no que acontece na rodada 3" | Antes de transições |
| **Twist Tease** | "Mas espera, porque eu não contei uma coisa..." | Quando a energia cai |
| **Stakes Raise** | "E eu acabei de DOBRAR o prêmio" | No meio do vídeo |
| **Countdown** | "Restam 3 participantes e 10 minutos" | Fase final |
| **Emotional Seed** | "Ele precisa desse dinheiro porque..." | Antes do clímax |

### Frequência

```
Vídeo de 10 min → 4-5 re-hooks (a cada 2 minutos)
Vídeo de 20 min → 8-10 re-hooks (a cada 2-3 minutos)
Short de 60 seg → 1-2 re-hooks (aos 15s e 40s)
```

| Content Ideation | Precisa gerar ideias de conteúdo viral |
| Re-hook System | Precisa manter retenção alta ao longo do vídeo |
