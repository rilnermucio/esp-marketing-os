# Park Howell - Frameworks

## 1. ABT Framework (And/But/Therefore)

O framework central de Howell. A "molécula do storytelling" que estrutura qualquer mensagem em 3 partes baseadas em contraste e resolução.

```
AND (E):        Estabelece o contexto, a situação, o ponto de partida
BUT (MAS):      Introduz o conflito, a tensão, o problema
THEREFORE (PORTANTO): Apresenta a resolução, a ação, o caminho
```

### Estrutura Detalhada

| Componente | Função | Efeito no Cérebro |
|------------|--------|-------------------|
| **AND** | Contexto e acordo | Cria rapport, o leitor concorda e relaxa |
| **BUT** | Conflito e surpresa | Ativa atenção, dopamina, curiosidade |
| **THEREFORE** | Resolução e ação | Satisfaz a necessidade de closure, motiva ação |

### Aplicação na Copy

**Template básico:**
```
[Contexto sobre a realidade do cliente] E [mais contexto que reforça].
MAS [o problema, o conflito, o obstáculo].
PORTANTO [a solução, o próximo passo, a ação].
```

**Exemplo para SaaS:**
```
AND: "Sua equipe trabalha duro todo dia e está comprometida
     com os resultados do negócio."

BUT: "Mas 40% do tempo deles é gasto em tarefas manuais que
     poderiam ser automatizadas."

THEREFORE: "Portanto, é hora de liberar sua equipe com
           automação inteligente para que façam o que importa."
```

### Os 3 Anti-Padrões a Evitar

| Padrão | Estrutura | Problema |
|--------|-----------|----------|
| **AAA** | And/And/And | Entediante, sem conflito, ignora-se |
| **BBB** | But/But/But | Negativo demais, sem solução, deprime |
| **DHY** | Despite/However/Yet | Acadêmico, frio, não motiva ação |

## 2. Story Cycle System (10 Passos)

Sistema completo de storytelling para marcas, baseado na Jornada do Herói adaptada para negócios.

### Os 10 Passos

```
1. BACKSTORY       → O contexto do seu cliente antes de te conhecer
2. DESIRE          → O que o cliente realmente quer
3. BARRIER         → O que impede o cliente de conseguir
4. ANTAGONIST      → O vilão (concorrência, status quo, medo)
5. MENTOR          → Sua marca como guia (empatia + expertise)
6. PLAN            → O caminho que você oferece (3-5 passos)
7. BATTLE          → O momento de decisão (CTA)
8. TRANSFORMATION  → A mudança que acontece
9. NEW NORMAL      → A vida depois da transformação
10. MORAL          → A lição ou verdade maior
```

### Quando Usar o Story Cycle Completo

| Situação | Use Story Cycle? |
|----------|-----------------|
| Brand story completo | Sim, todos os 10 passos |
| Vídeo institucional | Sim, adaptado |
| Landing page longa | Sim, como estrutura da página |
| Post de rede social | Não, use ABT simplificado |
| Pitch de elevador | Não, use ABT |
| Email curto | Não, use ABT |

## 3. Narrative Gym Framework

Exercício prático para treinar o "músculo narrativo" da equipe ou do empreendedor.

### Os 5 Exercícios

**Exercício 1: ABT de 1 Frase**
```
Escreva sua mensagem em exatamente 1 frase usando AND/BUT/THEREFORE.
Exemplo: "Empreendedores querem crescer, MAS a mensagem confusa
impede, PORTANTO precisam de storytelling."
```

**Exercício 2: ABT Expandido (3 Parágrafos)**
```
Parágrafo 1 (AND): Contexto completo
Parágrafo 2 (BUT): Problema detalhado
Parágrafo 3 (THEREFORE): Solução e CTA
```

**Exercício 3: Inversão do ABT**
```
Comece pelo THEREFORE (resultado desejado),
depois volte ao BUT (por que é difícil),
depois ao AND (o contexto). Útil para brainstorming.
```

**Exercício 4: ABT do Cliente**
```
Escreva o ABT na perspectiva do cliente, não da marca.
"Eu quero X, MAS enfrento Y, PORTANTO preciso de Z."
```

**Exercício 5: ABT Comparativo**
```
Escreva 3 versões do mesmo ABT:
- Versão emocional
- Versão racional
- Versão aspiracional
Escolha a mais impactante.
```

## 4. 5 Primal Elements of a Short Story

Framework para contar micro-histórias em formatos curtos (posts, emails, pitches).

### Os 5 Elementos

```
1. HERO       → Quem é o protagonista? (Seu cliente)
2. GOAL       → O que ele quer?
3. OBSTACLE   → O que impede?
4. MENTOR     → Quem ajuda? (Sua marca)
5. MORAL      → Qual a lição?
```

### Template de Micro-História

```
[HERO] queria [GOAL].
Mas [OBSTACLE] tornava isso quase impossível.
Então encontrou [MENTOR/SOLUÇÃO].
E agora [RESULTADO].
A verdade é: [MORAL].
```

## 5. Problema → Agitação → Resolução via ABT

Adaptação do clássico PAS (Problem/Agitate/Solve) usando a lente narrativa de Howell.

### Estrutura

```
AND (Problema contextualizado):
"Você construiu um negócio do zero. Trabalha 12 horas por dia.
 Investe em marketing."

BUT (Agitação com contraste):
"Mas quando alguém pergunta o que você faz, a explicação demora
 5 minutos e os olhos da pessoa vidram. E cada dia com uma
 mensagem confusa é um dia de oportunidades perdidas."

THEREFORE (Resolução com ação):
"Portanto, está na hora de simplificar sua história.
 Em 3 frases. Para que qualquer pessoa entenda e
 queira saber mais."
```

| Framework | Use quando... |
|-----------|---------------|
| ABT Framework | Precisa estruturar qualquer mensagem rápida e persuasiva |
| Story Cycle System | Está construindo uma narrativa de marca completa |
| Narrative Gym | Está treinando a equipe ou brainstormando mensagens |
| 5 Primal Elements | Precisa contar micro-histórias em formatos curtos |
| PAS via ABT | Está escrevendo copy de vendas com base narrativa |
