## Framework 2: A Pirâmide dos Estágios de Consciência

Adaptação de Suby dos estágios de Schwartz para o marketing digital moderno:

```
PIRÂMIDE DE CONSCIÊNCIA DO MERCADO:

    ┌──────────────┐
    │    3%        │ ← COMPRANDO AGORA
    │ "Já decidi"  │   (Seu concorrente está aqui)
    └──────────────┘
    ┌──────────────────┐
    │      17%         │ ← INFORMANDO
    │  "Estou          │   (Pesquisando opções)
    │   pesquisando"   │
    └──────────────────┘
    ┌──────────────────────┐
    │         20%          │ ← CONSCIENTE DO PROBLEMA
    │   "Tenho um          │   (Sabe que tem problema)
    │    problema"         │
    └──────────────────────┘
    ┌──────────────────────────┐
    │           60%            │ ← INCONSCIENTE
    │   "Não sei que           │   (Tem o problema mas
    │    tenho problema"       │    não sabe)
    └──────────────────────────┘

IMPLICAÇÃO:
Se você só anuncia para os 3%, está competindo pelo menor
pedaço do mercado contra todos os seus concorrentes.
```

## Framework 3: A Halo Strategy

Como criar conteúdo para cada estágio de consciência e nutrir prospects ao longo do tempo:

```
HALO STRATEGY — CRIAÇÃO DE CONTEÚDO:

CAMADA EXTERIOR (60% inconsciente):
  Conteúdo: Posts de problema (sem mencionar produto)
  Objetivo: Fazer o prospect identificar a dor
  Canal: Social media, artigos, SEO
  CTA: "Saiba mais sobre esse problema"

SEGUNDA CAMADA (20% consciente):
  Conteúdo: Conteúdo educativo sobre soluções
  Objetivo: Posicionar como autoridade
  Canal: Email, YouTube, webinars
  CTA: "Veja como outros resolveram"

TERCEIRA CAMADA (17% informando):
  Conteúdo: Comparativos e proof of concept
  Objetivo: Ganhar a comparação
  Canal: Retargeting, email avançado
  CTA: "Compare as opções"

NÚCLEO (3% comprando):
  Conteúdo: Oferta direta
  Objetivo: Fechar a venda
  Canal: Landing page, vendedor, webinar de venda
  CTA: "Compre agora"
```

## Framework 4: O Value Video (VSL) de Alta Conversão

Suby tem o framework de VSL mais testado do mercado australiano:

```
ESTRUTURA DO VALUE VIDEO:

00:00-00:30 — GANCHO IRRESISTÍVEL
  "Se você [avatar] quer [resultado], mas está
  lutando com [problema], então este vídeo vai
  mudar tudo para você."

00:30-02:00 — IDENTIFICAÇÃO DO PROBLEMA
  Descreve o problema com mais detalhe do que
  o próprio cliente saberia. Isso cria rapport.

02:00-04:00 — AGRAVAÇÃO
  O que acontece se o problema não for resolvido.
  Custos financeiros, emocionais, oportunidades perdidas.

04:00-06:00 — EPIFANIA / NOVA OPORTUNIDADE
  "E então eu descobri que..."
  A virada que muda tudo.

06:00-10:00 — APRESENTAÇÃO DA SOLUÇÃO
  Como funciona. Por que é diferente.
  Framework nomeado.

10:00-15:00 — PROVA SOCIAL EM CASCATA
  3-5 casos de sucesso específicos com números.

15:00-17:00 — STACK DE VALOR
  Tudo que está incluído + valor individual de cada item.

17:00-18:00 — REVELAÇÃO DE PREÇO
  O preço total comparado ao stack de valor.

18:00-20:00 — GARANTIA
  Remove o risco completamente.

20:00-21:00 — URGÊNCIA/ESCASSEZ REAL
  Razão concreta para agir agora.

21:00-22:00 — CTA MÚLTIPLOS
  "Clique no botão abaixo, preencha o formulário..."
```

## Framework 5: O Sistema HDIC

O método de 3 etapas de Suby para geração previsível de clientes:

```
H — HORDE (Atrair)
  Atrair uma quantidade massiva de prospects qualificados
  usando paid traffic + conteúdo de consciência

  FERRAMENTAS: Facebook Ads, Google Ads, YouTube Ads
  MÉTRICA: CPL (Custo por Lead)
  META: Volume + qualidade (não apenas volume)

D — DIRECT (Direcionar)
  Guiar prospects por um processo específico de nutrição
  que aumenta a temperatura (interesse + confiança)

  FERRAMENTAS: Email marketing, retargeting, VSL
  MÉTRICA: Taxa de engajamento, abertura, clique
  META: Mover de "frio" para "quente"

C — CONVERT (Converter)
  Converter prospects quentes em clientes pagantes
  com oferta irresistível e processo de vendas otimizado

  FERRAMENTAS: Página de vendas, webinar, ligação de vendas
  MÉTRICA: Taxa de conversão, CAC, ROAS
  META: Custo de aquisição abaixo do LTV
```

### Métricas por Etapa HDIC

| Etapa | Métrica Primária | Benchmark |
|-------|-----------------|-----------|
| **Horde** | CPL (Custo por Lead) | Varia por nicho |
| **Direct** | Taxa de abertura email | > 25% |
| **Convert** | ROAS | > 3x (mínimo) |
| **Sistema completo** | CAC vs LTV | LTV > 3× CAC |
