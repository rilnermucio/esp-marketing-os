## A Ciência da Retenção

### O Que os Algoritmos Realmente Medem

```
HIERARQUIA DE SINAIS DE VÍDEO

TIER 1 - Sinais Críticos (Maior Peso)
├── Average View Duration (AVD)
│   └── Quanto tempo em média assistem
├── Average Percentage Viewed (APV)
│   └── % do vídeo assistido
├── Watch Time Total
│   └── Soma de todos os minutos assistidos
└── Retention Curve Shape
    └── Formato da curva de retenção

TIER 2 - Sinais de Engajamento
├── Click-Through Rate (CTR)
│   └── % que clica após ver thumbnail
├── Likes / Dislikes ratio
├── Comments
├── Shares
└── Subscribers gained

TIER 3 - Sinais de Satisfação
├── Session time (tempo na plataforma após seu vídeo)
├── Return viewers
├── Notification clicks
└── Playlist adds
```

### A Fórmula do Watch Time

```
WATCH TIME = Impressões × CTR × AVD

Para maximizar:
1. IMPRESSÕES: Algoritmo mostra mais se os outros sinais forem bons
2. CTR: Thumbnail + Título = Clique
3. AVD: Retenção = Quanto tempo ficam

INSIGHT CRUCIAL:
Um vídeo de 10min com 70% de retenção (7min AVD)
SUPERA um vídeo de 20min com 40% de retenção (8min AVD)
Por quê? Melhor curva de retenção = mais distribuição
```

### Re-hooks: A Técnica dos 30 Segundos

```
RE-HOOK = Novo gancho a cada 30-60 segundos

TIPOS DE RE-HOOK:

1. VERBAL RE-HOOK
   "Mas fica ainda melhor..."
   "Isso é só o começo..."
   "Espera, porque..."
   "O que aconteceu depois foi..."

2. VISUAL RE-HOOK
   - Mudança de ângulo
   - Zoom dramático
   - B-roll inesperado
   - Gráfico/animação

3. AUDIO RE-HOOK
   - Efeito sonoro
   - Mudança de música
   - Silêncio dramático
   - Intensificação

4. INFORMAÇÃO RE-HOOK
   - Nova revelação
   - Plot twist
   - Dado surpreendente
   - Teaser do que vem

MAPA DE RE-HOOKS (vídeo de 10min):
0:00 - Hook principal
0:30 - Re-hook 1 (transição para conteúdo)
1:30 - Re-hook 2 (promessa específica)
3:00 - Re-hook 3 (pattern interrupt)
5:00 - Re-hook 4 (meio do vídeo - CRÍTICO)
7:00 - Re-hook 5 (teaser do final)
9:00 - Re-hook 6 (payoff + próximo vídeo)
```

### Análise de Curva de Retenção

```
DIAGNÓSTICO DE PROBLEMAS

QUEDA ABRUPTA NO INÍCIO (0-30s):
├── Problema: Hook fraco
├── Solução: Reescrever abertura
└── Teste: A/B com diferentes hooks

QUEDA GRADUAL CONSTANTE:
├── Problema: Falta de re-hooks
├── Solução: Adicionar pattern interrupts
└── Teste: Mais cortes, mais energia

QUEDA EM PONTOS ESPECÍFICOS:
├── Problema: Momento boring identificado
├── Solução: Cortar ou acelerar esse trecho
└── Teste: Remover completamente

PICO DE REWATCH (volta atrás):
├── Sinal positivo: Conteúdo denso/confuso
├── Ação: Simplificar ou dividir em partes
└── Oportunidade: Fazer vídeo sobre esse trecho

QUEDA ANTES DO CTA:
├── Problema: CTA muito no final
├── Solução: CTA no meio também
└── Teste: Múltiplos CTAs distribuídos
```

## Psicologia do Vídeo

### Por Que Vídeo é Mais Poderoso

```
VÍDEO ATIVA MAIS ÁREAS DO CÉREBRO

┌─────────────────────────────────────────────────┐
│              PROCESSAMENTO DE VÍDEO             │
├─────────────────────────────────────────────────┤
│  VISUAL (50% do cérebro)                        │
│  ├── Movimento captura atenção involuntária     │
│  ├── Rostos ativam reconhecimento social        │
│  └── Cores e contraste direcionam foco          │
├─────────────────────────────────────────────────┤
│  AUDITIVO                                       │
│  ├── Voz humana gera conexão emocional          │
│  ├── Música modula estado emocional             │
│  └── Efeitos sonoros amplificam impacto         │
├─────────────────────────────────────────────────┤
│  EMOCIONAL (Neurônios Espelho)                  │
│  ├── Expressões faciais são contagiosas         │
│  ├── Sentimos o que vemos outros sentindo       │
│  └── Histórias ativam empatia                   │
├─────────────────────────────────────────────────┤
│  COGNITIVO                                      │
│  ├── Narrativa facilita memorização             │
│  ├── Demonstração > Explicação                  │
│  └── Múltiplos canais = maior retenção          │
└─────────────────────────────────────────────────┘

DADOS:
• Retenção de informação: Texto 10% | Vídeo 95%
• Processamento: Vídeo é 60.000x mais rápido que texto
• Preferência: 72% preferem vídeo para aprender produto
```

### O Poder do Rosto Humano

```
NEUROCIÊNCIA DO ROSTO EM VÍDEO

O cérebro humano tem uma área dedicada (FFA - Fusiform Face Area)
que processa rostos automaticamente.

IMPLICAÇÕES PRÁTICAS:

1. TALKING HEAD FUNCIONA
   - Rostos na thumbnail aumentam CTR em 30%
   - Contato visual cria conexão
   - Expressões faciais transmitem emoção

2. EXPRESSÕES AMPLIFICADAS
   - Câmera "achata" emoções
   - Exagere 30% para parecer natural
   - Sobrancelhas são cruciais (expressam emoção)

3. POSICIONAMENTO
   - Regra dos terços: olhos no terço superior
   - Espaço para texto: deixe "ar" ao redor
   - Direção do olhar: para onde olha, audiência olha

4. ENERGIA FACIAL
   - Rosto parado = morte do vídeo
   - Micro-expressões mantêm atenção
   - Sorrir (genuinamente) aumenta likability
```

### Storytelling Visual: A Jornada em Vídeo

```
ESTRUTURA NARRATIVA PARA VÍDEO

ATO 1: SETUP (10-15% do vídeo)
├── Mundo comum (situação atual)
├── Personagem (você ou o espectador)
├── Problema/desejo introduzido
└── INCITING INCIDENT: Por que assistir?

ATO 2: CONFRONTO (70-80% do vídeo)
├── Obstáculos e desafios
├── Tentativas e falhas
├── Escalada de stakes
├── PONTO DE VIRADA: Descoberta/solução
└── Subida para o clímax

ATO 3: RESOLUÇÃO (10-15% do vídeo)
├── Clímax: Maior momento de tensão/revelação
├── Resolução: Problema resolvido
├── Novo mundo: Estado transformado
└── CTA: O que fazer agora

APLICAÇÃO EM VÍDEO EDUCATIVO:
Setup: "Você já tentou X e não funcionou?"
Confronto: "A maioria faz errado porque Y..."
Resolução: "Aqui está o método que funciona..."
```

### Emoções que Retêm

```
MAPA EMOCIONAL DO VÍDEO

EMOÇÕES QUE MANTÊM ASSISTINDO:

1. CURIOSIDADE
   → "O que vai acontecer?"
   → Open loops, mistério, revelações graduais

2. ANTECIPAÇÃO
   → "Estou esperando algo bom"
   → Promessas, countdowns, build-ups

3. SURPRESA
   → "Não esperava isso!"
   → Plot twists, revelações, subversão de expectativa

4. AWE (Admiração)
   → "Isso é incrível!"
   → Resultados impressionantes, visuais espetaculares

5. TENSÃO
   → "Vai dar certo?"
   → Stakes, conflito, incerteza

EMOÇÕES QUE FAZEM SAIR:

1. CONFUSÃO
   → "Não estou entendendo"
   → Explicações confusas, pulos de lógica

2. TÉDIO
   → "Não está acontecendo nada"
   → Ritmo lento, redundância, filler

3. FRUSTRAÇÃO
   → "Cadê o que prometeu?"
   → Clickbait, enrolação, falta de payoff
```

## Anatomia do Hook Perfeito

### Os 3 Segundos de Ouro

```
FRAMEWORK: HOOK = Interrupção + Promessa + Prova

0-1 SEGUNDO: INTERRUPÇÃO
├── Pattern break visual ou sonoro
├── Movimento inesperado
├── Afirmação chocante
└── Objetivo: Parar o scroll / Capturar atenção

1-2 SEGUNDOS: PROMESSA
├── O que vão ganhar assistindo
├── Benefício implícito ou explícito
├── Curiosidade amplificada
└── Objetivo: Criar razão para continuar

2-3 SEGUNDOS: PROVA
├── Por que você é credível
├── Preview do resultado
├── Social proof rápido
└── Objetivo: Justificar o investimento de tempo
```

### Tipos de Hook por Objetivo

```
HOOKS POR CATEGORIA

CURIOSITY HOOKS (Geram curiosidade)
├── "A razão número 1 que [problema] é..."
├── "O que ninguém te conta sobre [tema]..."
├── "Existe um segredo sobre [tema] que..."
├── "Eu descobri algo sobre [tema] que..."
└── Por que funciona: Lacuna de informação

CONTRARIAN HOOKS (Desafiam o status quo)
├── "Pare de fazer [ação comum] agora"
├── "Tudo que você sabe sobre [tema] está errado"
├── "[Conselho popular] é o pior conselho"
├── "Por que [coisa popular] não funciona"
└── Por que funciona: Dissonância cognitiva

RESULT HOOKS (Mostram resultados)
├── "Como eu fui de [antes] para [depois]"
├── "Isso me fez [resultado] em [tempo]"
├── "O método exato que gerou [resultado]"
├── "[Número impressionante] usando apenas [método]"
└── Por que funciona: Prova social + aspiração

STORY HOOKS (Contam história)
├── "Era [data], quando [evento]..."
├── "Tudo mudou quando eu descobri..."
├── "Eu nunca contei isso publicamente, mas..."
├── "A história de como [resultado improvável]"
└── Por que funciona: Narrativa + emoção

FEAR HOOKS (Ativam medo)
├── "Você está cometendo esse erro sem saber"
├── "Isso está destruindo sua [área]"
├── "Se você faz [ação], pare agora"
├── "O perigo escondido em [coisa comum]"
└── Por que funciona: Instinto de sobrevivência

AUTHORITY HOOKS (Estabelecem autoridade)
├── "Depois de [tempo/experiência] fazendo [área]..."
├── "Como [profissão] por [anos], eu descobri..."
├── "[Número] de clientes me ensinaram que..."
├── "A técnica que uso com [clientes importantes]"
└── Por que funciona: Transferência de credibilidade
```

### Fórmulas de Hook Testadas

```
FÓRMULAS PRONTAS (Preencha os espaços)

1. "Se você quer [resultado desejado], os próximos [tempo]
    vão mudar sua vida"

2. "A maioria das pessoas fazem [ação errada].

7. "O erro de [valor] que eu cometi,

8. "Se [situação], para tudo e assiste isso"

9. "[Número] segundos que vão [resultado]"

10. "Eu não devia estar mostrando isso, mas..."

## Edição e Ritmo Avançado

### O Ritmo que Retém

```
TEORIA DO RITMO EM VÍDEO

O cérebro humano se adapta a padrões.
Para manter atenção, você precisa quebrar padrões.

REGRA DOS 3-7 SEGUNDOS:
├── A cada 3-7 segundos, algo deve mudar
├── Corte, ângulo, movimento, gráfico, som
└── Previsibilidade = tédio = abandono

MAPA DE RITMO POR FORMATO:

SHORTS/REELS/TIKTOK (Alta Energia):
├── Corte a cada 1-2 segundos
├── Movimento constante
├── Texto na tela alternando
└── Sem pausas

YOUTUBE EDUCATIVO (Média Energia):
├── Corte a cada 3-5 segundos
├── Alternância talking head ↔ b-roll
├── Gráficos para conceitos
└── Pausas estratégicas

DOCUMENTÁRIO/STORYTELLING (Energia Variável):
├── Cortes mais longos (5-10s)
├── Ritmo segue a emoção
├── Build-ups e payoffs
└── Silêncios dramáticos permitidos
```

### Técnicas de Edição Pro

```
TÉCNICAS QUE AUMENTAM RETENÇÃO

1. JUMP CUTS
   ├── Cortar pausas, "uhhs", hesitações
   ├── Manter apenas o essencial
   └── Cria sensação de velocidade

2. J-CUT & L-CUT
   ├── J-Cut: Áudio começa antes do visual
   ├── L-Cut: Áudio continua após corte visual
   └── Cria transições suaves e profissionais

3. MATCH CUT
   ├── Movimento similar entre dois clips
   ├── Forma similar em frames diferentes
   └── Cria continuidade visual

4. ZOOM DINÂMICO
   ├── Slow zoom in: Aumenta intimidade/tensão
   ├── Slow zoom out: Revela contexto
   ├── Punch in: Ênfase em ponto importante
   └── Ken Burns: Movimento em imagens estáticas

5. SPEED RAMPING
   ├── Acelerar partes menos importantes
   ├── Desacelerar momentos-chave
   └── Cria drama e impacto

6. SPLIT SCREEN
   ├── Antes/depois simultâneo
   ├── Comparações diretas
   └── Múltiplas perspectivas
```

### A Camada Sonora

```
DESIGN DE ÁUDIO PARA RETENÇÃO

1. MÚSICA DE FUNDO
   ├── Energia compatível com conteúdo
   ├── Build-ups em momentos-chave
   ├── Drops em revelações
   ├── Volume: 10-20% do volume da voz
   └── Mudança de música = mudança de seção

2. EFEITOS SONOROS (SFX)
   ├── Whoosh: Transições
   ├── Pop: Aparição de texto/gráfico
   ├── Ding: Pontos importantes
   ├── Boom: Revelações/impacto
   └── Silence: Antes de momento importante

3. PROCESSAMENTO DE VOZ
   ├── Compressão para consistência
   ├── EQ para clareza
   ├── De-essing se necessário
   └── Noise reduction sutil

REGRA DE OURO:
"Se o espectador notar a música, está muito alta.
Se não sentir a emoção, está muito baixa."
```

### Texto na Tela (Motion Graphics)

```
QUANDO USAR TEXTO NA TELA

1. REFORÇO DE PONTO-CHAVE
   ├── Aparece quando fala
   ├── Desaparece após 2-3 segundos
   └── Fonte legível, tamanho adequado

2. CALL-OUTS
   ├── Setas, círculos, destaques
   ├── Direciona atenção
   └── Especialmente útil em tutoriais

3. CHAPTERING (Capítulos)
   ├── "PASSO 1: [Nome]"
   ├── Orienta o espectador
   └── Permite pular seções

4. NUMBERS/STATS
   ├── Dados importantes em texto
   ├── Mais memorável
   └── Prova visual de claims

5. ACESSIBILIDADE
   ├── Legendas sempre
   ├── 85% assistem sem som
   └── Aumenta watch time em 12%

REGRAS DE TEXTO:
• Sans-serif para telas (Montserrat, Poppins)
• Contraste mínimo de 4.5:1
• Tempo na tela: 3-5 segundos
• Animação de entrada/saída
• Não cobrir rosto
```

### VSL - Video Sales Letter (Avançado)

O VSL é o formato mais lucrativo de vídeo. Esta seção expande o conhecimento para criar VSLs de alta conversão.

#### A Ciência por Trás do VSL

```
POR QUE VSL CONVERTE MAIS

CONTROLE DA EXPERIÊNCIA:
├── Você controla o ritmo
├── Você controla a narrativa
├── Não há distração
└── Prospect assiste até estar pronto

ENGAJAMENTO MULTISENSORIAL:
├── Visual + Auditivo = maior retenção
├── Emoções transmitidas pela voz
├── Demonstração visual de prova
└── Mais persuasivo que texto

TEMPO DE EXPOSIÇÃO:
├── Mais tempo com sua mensagem
├── Mais objeções respondidas
├── Maior conexão emocional
└── Mais momentum para ação
```

#### Fórmulas de VSL que Convertem

```
FÓRMULA 1: PAS (Problem-Agitate-Solve)
────────────────────────────────────────
PROBLEMA: Identifique a dor
AGITAÇÃO: Amplifique as consequências
SOLUÇÃO: Apresente sua oferta

Exemplo:
"Você está cansado de trabalhar 12h/dia..."
"E o pior é que nem consegue ver sua família..."
"É por isso que criei o Método [X]..."

FÓRMULA 2: QUEST (Qualify-Understand-Educate-Stimulate-Transition)
────────────────────────────────────────
QUALIFY: Para quem é
UNDERSTAND: Mostre que entende a dor
EDUCATE: Ensine algo valioso
STIMULATE: Crie desejo
TRANSITION: Leve para oferta

FÓRMULA 3: HERO'S JOURNEY
────────────────────────────────────────
MUNDO COMUM: Onde você estava
CHAMADO: O problema aparece
RECUSA: Dúvidas iniciais
MENTOR: Quem/o que te ajudou
LIMIAR: Decisão de agir
PROVAS: Obstáculos superados
APROXIMAÇÃO: Chegando perto
PROVAÇÃO: Maior desafio
RECOMPENSA: Resultado alcançado
RETORNO: Voltar para ajudar outros
ELIXIR: O conhecimento que compartilha

FÓRMULA 4: STAR-STORY-SOLUTION
────────────────────────────────────────
STAR: Apresente protagonista (você ou cliente)
STORY: Conte a jornada com conflito
SOLUTION: Revele seu produto como a ponte
```

## Thumbnails que Convertem

### A Ciência do Clique

```
CTR = f(Curiosidade + Contraste + Clareza)

DADOS REAIS:
• Thumbnail + Título = 50% do sucesso de um vídeo
• 2 segundos para decidir clicar ou não
• Rostos aumentam CTR em 30%
• Contraste de cores aumenta CTR em 25%
• Texto legível em mobile é OBRIGATÓRIO
```

### Elementos de Thumbnail de Alta Conversão

```
FRAMEWORK THUMBNAIL

┌─────────────────────────────────────────────────┐
│                                                 │
│   [ROSTO COM EXPRESSÃO]        [ELEMENTO       │
│   - Emoção exagerada           VISUAL]         │
│   - Olhos visíveis             - Objeto        │
│   - Direção do olhar →         - Resultado     │
│                                - Contraste     │
│                                                 │
│   [TEXTO (3-4 PALAVRAS MÁX)]                   │
│   - Complementa título                         │
│   - Não repete título                          │
│   - Cria curiosidade                           │
│                                                 │
│   [BRANDING SUTIL]                             │
│   - Cor consistente                            │
│   - Reconhecível mas não dominante             │
│                                                 │
└─────────────────────────────────────────────────┘
```

### Fórmulas de Thumbnail por Tipo

```
ANTES/DEPOIS:
├── Lado esquerdo: "antes" (menos atrativo)
├── Lado direito: "depois" (aspiracional)
├── Seta ou divisão clara
└── Expressão de surpresa/satisfação

LISTICLE ("7 coisas...", "5 erros..."):
├── Número grande e destacado
├── Objeto representativo
├── Expressão de descoberta
└── Um item revelado (cria curiosidade)

TUTORIAL/HOW-TO:
├── Resultado final visível
├── Ferramenta/processo mostrado
├── Setas direcionais
└── "Antes → Depois" implícito

REAÇÃO/OPINIÃO:
├── Rosto com forte emoção
├── Objeto/pessoa sendo avaliado
├── Texto com opinião ("WOW", "RUIM", "🔥")
└── Alto contraste emocional

STORYTELLING:
├── Momento dramático da história
├── Cenário que cria curiosidade
├── Personagem em ação
└── Mistério visual
```

### Erros Comuns em Thumbnails

```
EVITAR:

❌ Muito texto (mais de 4-5 palavras)
❌ Texto ilegível em mobile
❌ Imagem genérica sem emoção
❌ Cores apagadas ou sem contraste
❌ Rosto sem expressão
❌ Clickbait que não entrega
❌ Inconsistência de branding
❌ Resolução baixa
❌ Elementos cortados nas bordas
❌ Repetir exatamente o título
```

### Teste A/B de Thumbnails

```
PROCESSO DE OTIMIZAÇÃO

1. CRIAR 3-5 VARIAÇÕES
   ├── Diferentes expressões
   ├── Diferentes textos
   ├── Diferentes cores
   └── Diferentes composições

2. TESTAR NAS PRIMEIRAS 24-48h
   ├── YouTube permite trocar thumbnail
   ├── Monitorar CTR por variação
   └── TubeBuddy/VidIQ para análise

3. ITERAR BASEADO EM DADOS
   ├── CTR > 5% = Bom
   ├── CTR > 8% = Ótimo
   ├── CTR > 10% = Excepcional
   └── Sempre testar novas ideias
```

## Métricas e Analytics de Vídeo

### Framework de Métricas por Formato

#### YouTube Long-Form

| Métrica | Ruim | Bom | Ótimo | O que indica |
|---------|------|-----|-------|--------------|
| CTR (Thumbnail) | <3% | 5-8% | >10% | Atratividade do pacote |
| Retenção 30s | <70% | 80-90% | >90% | Qualidade do hook |
| Retenção média | <40% | 50-60% | >70% | Qualidade geral |
| AVD vs duração | <30% | 40-60% | >70% | Eficiência do conteúdo |
| Likes/views | <3% | 4-6% | >7% | Satisfação |
| Comments/views | <0.5% | 1-2% | >3% | Engajamento profundo |
| Subs/views | <0.5% | 1-2% | >3% | Conversão de canal |

```
MÉTRICAS AVANÇADAS YOUTUBE

1. IMPRESSIONS-TO-VIEWS RATIO
   └── Quantas impressões você precisa para 1 view
   └── Benchmark: <20 impressões por view

2. CLICK-THROUGH RATE BY SOURCE
   ├── Browse features: 3-8%
   ├── Suggested videos: 2-6%
   ├── Search: 5-15%
   └── External: 1-3%

3. RETENTION CLIFF ANALYSIS
   └── Identificar pontos exatos de queda
   └── Cada 1% de retenção = mais distribuição

4. SESSION TIME IMPACT
   └── Seu vídeo mantém pessoas no YouTube?
   └── Algoritmo prioriza quem aumenta session time

5. SUBSCRIBER VELOCITY
   └── Subs ganhos / Subs perdidos
   └── Ratio > 1 = crescimento saudável

   └── <30% = hook funcionando
   └── >50% = hook precisa melhorar

2. COMPLETION RATE
   └── % que assiste até o final
   └── Correlação direta com FYP/distribuição

3. SOUND PAGE CLICKS
   └── Se usam seu áudio = viral potential

4. STITCH/DUET RATE
   └── Engajamento criativo = multiplicador

5. SAVE RATE
   └── Salvar = valor percebido alto
   └── Benchmark: >1% = ótimo

```
ANÁLISE DE FUNIL VSL

PLAY RATE → ENGAGEMENT → WATCH → CLICK → CONVERSION

└── Hook fraco, promessa não ressoa

Se queda no meio:
└── Problema/solução não convincente

Se queda antes da oferta:
└── Prova social insuficiente

Se pouco clique:
└── Oferta não clara ou preço alto percebido

Se baixa conversão pós-clique:
└── Checkout problemático ou objeções não resolvidas

### Princípios Universais dos Top Video Creators

```
1. HOOK OU MORRA
   - Primeiros segundos definem tudo
   - Nunca começar com "oi, eu sou..."
   - Ação, promessa ou choque primeiro

2. CADA SEGUNDO TEM CUSTO
   - Se não adiciona, corta
   - Tempo do viewer é precioso
   - Ruthless editing

3. RETENÇÃO > VIEWS
   - Algoritmo prioriza watch time
   - Melhor 10min com 70% retenção que 20min com 40%
   - Qualidade > quantidade

4. THUMBNAIL + TÍTULO = 50%
   - Criar antes do vídeo
   - Testar múltiplas versões
   - Não mentir (clickbait destrói canal)

5. CONSISTÊNCIA COMPOSTA
   - 1 vídeo/semana por anos > Surtos de produção
   - Algoritmo recompensa consistência
   - Audiência confia em quem aparece

6. CONHEÇA SEU AVATAR
   - Um vídeo, uma pessoa
   - Falar COM, não PARA
   - Resolver UM problema bem

7. DADOS INFORMAM, NÃO DITAM
   - Analytics mostram O QUE
   - Criatividade descobre O PORQUÊ
   - Testar hipóteses baseadas em dados

**Regra de precedência:** Se o usuário mencionar um clone explicitamente, usar esse. Se não mencionar, aplicar a tabela abaixo. Para Reels, o `clone-manifest.yaml` é a fonte de verdade: `mrbeast` é o padrão, `brunson` é ativado quando o objetivo for venda/conversão.

| Reels de venda / funil | **Brunson** | **Hormozi** | Hook-story-offer, escassez, conversão |

| Vídeo Institucional | **Ogilvy** |, | Sofisticado, brand-driven |

### Como o Clone Afeta o Roteiro

**Hormozi no VSL:**
- Hook com número específico: "Este método gerou R$2.3M em 14 meses"
- Prova com dados concretos
- CTA com lógica matemática (valor > preço)

**Brunson no Reels:**
- Hook magnético nos primeiros 2s
- Storytelling compacto (problema → virada → resultado)
- CTA com framework de escassez

**Ogilvy no YouTube:**
- Abertura com fato surpreendente baseado em pesquisa
- Desenvolvimento com autoridade e exemplos
- Fechamento com insight acionável

### Como Ativar

Mencione o clone no briefing: "criar roteiro de Reels com estilo Brunson para nicho fitness".

O sistema carrega automaticamente a persona, voz, frameworks e exemplos do clone selecionado.
