# 📅 Workflow: Calendário Editorial Mensal

> Workflow do Marketing OS. A numeração e os padrões de orquestração canônicos estão em `skills/marketing-os/SKILL.md` (seção "Padrões de Orquestração").

Sistema completo para planejamento e execução de conteúdo mensal.

---

## VISÃO GERAL DO MÊS

### Estrutura de Pilares de Conteúdo

```
PILAR 1: EDUCACIONAL (40%)
→ Ensina algo prático
→ Resolve problemas
→ Tutoriais e how-to

PILAR 2: INSPIRACIONAL (25%)
→ Histórias de sucesso
→ Motivação
→ Bastidores e jornada

PILAR 3: ENTRETENIMENTO (20%)
→ Trends adaptadas
→ Humor do nicho
→ Conteúdo leve

PILAR 4: CONVERSÃO (15%)
→ Ofertas e CTAs
→ Prova social
→ Conteúdo de venda
```

---

## 📋 PLANEJAMENTO MENSAL

### Semana -1: Planejamento (Última semana do mês anterior)

**Dia 1-2: Análise**
```
□ Revisar métricas do mês anterior
□ Identificar top 5 conteúdos
□ Identificar bottom 5 conteúdos
□ Analisar comentários e perguntas frequentes
□ Verificar tendências do nicho
```

**Dia 3-4: Ideação**
```
□ Brainstorm de 30+ ideias
□ Pesquisa de keywords e trends
□ Mapear datas comemorativas do mês
□ Verificar lançamentos/eventos planejados
□ Identificar ganchos de notícias
```

**Dia 5-6: Organização**
```
□ Categorizar ideias por pilar
□ Definir formatos para cada ideia
□ Priorizar por potencial de engajamento
□ Distribuir no calendário
□ Criar briefings de produção
```

**Dia 7: Aprovação**
```
□ Revisar calendário completo
□ Validar alinhamento com objetivos
□ Aprovar ou ajustar
□ Compartilhar com equipe (se aplicável)
```

## 🗓️ TEMPLATE DE CALENDÁRIO SEMANAL

### Segunda-feira: EDUCACIONAL
```
Manhã:
- Post de feed: Tutorial/How-to
- Stories: Dica rápida do dia

Noite:
- Reels/TikTok: Micro-tutorial
- Stories: Bastidores da criação
```

### Terça-feira: INSPIRACIONAL
```
Manhã:
- Post de feed: História/Case
- Stories: Motivação matinal

Noite:
- Stories: Interação (enquete sobre o tema)
```

### Quarta-feira: EDUCACIONAL
```
Manhã:
- Carrossel educativo (5-10 slides)
- Stories: Passo a passo

Noite:
- Reels/TikTok: Dica #2 da semana
- Stories: Q&A
```

### Quinta-feira: ENTRETENIMENTO
```
Manhã:
- Post de feed: Trend adaptada ou meme do nicho
- Stories: Conteúdo leve

Noite:
- Reels/TikTok: Trend/POV
- Stories: Interação
```

### Sexta-feira: CONVERSÃO
```
Manhã:
- Post de feed: Prova social/Depoimento
- Stories: Oferta ou CTA

Noite:
- Stories: Resultados da semana
- Reels/TikTok (opcional): Resumo
```

### Sábado: REPOUSO/LEVE
```
- Stories: Bastidores pessoais
- Repost de conteúdo (opcional)
- Interação com comunidade
```

### Domingo: PLANEJAMENTO
```
- Revisão da semana
- Preparação da próxima
- Conteúdo evergreen (opcional)
```

## 📊 DISTRIBUIÇÃO MENSAL POR FORMATO

### Instagram
```
FEED (12-16 posts/mês):
- 4 Carrosséis educativos
- 4 Posts estáticos (citações, dicas)
- 2 Depoimentos/prova social
- 2 Ofertas/CTA

REELS (12-20/mês):
- 8 Tutoriais/dicas rápidas
- 4 Trends adaptadas
- 4 POV/Storytime

STORIES (diário):
- Mínimo 5-10 por dia
- Mix de interação, bastidores, conteúdo
```

### LinkedIn
```
POSTS (8-12/mês):
- 4 Textos longos (artigos)
- 4 Carrosséis adaptados
- 2 Depoimentos/cases
- 2 Reflexões pessoais
```

### YouTube
```
VÍDEOS LONGOS (4-8/mês):
- 2-4 Tutoriais completos
- 1-2 Entrevistas/conversas
- 1 Vlog/bastidores

SHORTS (8-12/mês):
- Cortes dos vídeos longos
- Dicas rápidas originais
```

### TikTok
```
VÍDEOS (20-30/mês):
- 10 Tutoriais/dicas
- 8 Trends
- 5 POV/Storytime
- 5 Conteúdo original
```

### Email
```
NEWSLETTERS (4-8/mês):
- 1-2 por semana
- Mix de valor + promoção
```

```

```


### Sexta-feira
```
□ Publicar conteúdo de conversão
□ Análise rápida da semana

```

```
□ Manter Stories (opcional)
□ Planejar próxima semana
□ Criar conteúdo batch (opcional)
□ Descansar e recarregar
```

## 🔄 RECICLAGEM DE CONTEÚDO

### Conteúdo Evergreen para Repostar
```
A cada 3-6 meses, repostar:
- Top performers (com ajustes)
- Tutoriais sempre relevantes
- Posts de alta conversão
```

### Transformação de Formatos
```
Artigo de blog → Carrossel + Thread + Newsletter
Vídeo longo → 5-10 Shorts/Reels
Podcast → Posts de citação + Clipes
Webinar → Série de posts educativos
```

### Atualização de Conteúdo
```
□ Atualizar dados desatualizados
□ Melhorar design de posts antigos
□ Adicionar CTAs mais efetivos
□ Testar novos hooks
```
