# Workflow: Produção em Batch de Conteúdo

> Workflow do Marketing OS. A numeração e os padrões de orquestração canônicos estão em `skills/marketing-os/SKILL.md` (seção "Padrões de Orquestração").

Processo completo para produzir Reels, Carrosséis e Cards Únicos em lote, maximizando eficiência e qualidade.

---

## Visão Geral do Batch Production

```
O QUE É BATCH PRODUCTION?

Produzir múltiplos conteúdos de uma só vez, ao invés de
criar um por vez conforme a necessidade.

BENEFÍCIOS:
├── Economia de tempo (setup único)
├── Consistência de qualidade
├── Redução de estresse diário
├── Mais tempo para engajamento
├── Possibilidade de revisar antes de postar
└── Flexibilidade para aproveitar trends

META SEMANAL SUGERIDA:
┌─────────────────────────────────────┐
│  3 Reels + 2 Carrosséis + 5 Cards  │
│  = 1 sessão de batch production    │
└─────────────────────────────────────┘
```

## Cronograma de Produção

### Dia 1: Planejamento (2-3 horas)

```
MANHÃ OU NOITE ANTERIOR

09:00 - 09:30 │ PESQUISA DE TENDÊNCIAS
              ├── Scrollar Reels/TikTok (15 min)
              ├── Anotar áudios trending
              ├── Identificar formatos em alta
              └── Salvar referências

09:30 - 10:00 │ BRAINSTORM DE IDEIAS
              ├── Listar 20+ ideias brutas
              ├── Conectar com trends encontradas
              ├── Verificar perguntas da audiência
              └── Revisar conteúdos anteriores (top performers)

10:00 - 10:30 │ SELEÇÃO E PRIORIZAÇÃO
              ├── Escolher 10 melhores ideias
              ├── Distribuir por formato:
              │   ├── 3 ideias → Reels
              │   ├── 2 ideias → Carrossel
              │   └── 5 ideias → Card Único
              └── Verificar mix de pilares de conteúdo

10:30 - 11:00 │ CRIAÇÃO DE ROTEIROS
              ├── Escrever hooks para cada conteúdo
              ├── Estruturar Reels (hook > setup > conteúdo > CTA)
              ├── Planejar slides do carrossel
              └── Rascunhar captions

11:00 - 11:30 │ PREPARAÇÃO DE ASSETS
              ├── Listar elementos visuais necessários
              ├── Preparar figurino/cenário (Reels)
              ├── Escolher templates (Carrossel/Card)
              └── Baixar áudios/músicas necessários
```

### Dia 2: Produção de Reels (3-4 horas)

```
SESSÃO DE GRAVAÇÃO

ANTES DE COMEÇAR:
├── Carregar celular 100%
├── Limpar memória do celular
├── Preparar figurinos (todos de uma vez)
├── Montar cenário/iluminação
├── Testar áudio do ambiente
└── Ter água por perto

SETUP DE GRAVAÇÃO BÁSICO:
┌─────────────────────────────────────┐
│         💡 LUZ (janela/ring light)  │
│              ↓                      │
│         📱 CÂMERA                   │
│           (altura dos olhos)        │
│              ↓                      │
│         🧍 VOCÊ                     │
│    (1-2m de distância da câmera)   │
│              ↓                      │
│         🎤 ÁUDIO                    │
│    (microfone lapela ou celular)   │
└─────────────────────────────────────┘

ROTEIRO DE GRAVAÇÃO:

14:00 - 14:15 │ SETUP
              ├── Montar equipamento
              ├── Testar iluminação
              ├── Verificar enquadramento
              └── Fazer take de teste

14:15 - 14:45 │ REEL 1
              ├── Revisar roteiro (2 min)
              ├── Gravar 3-5 takes (15 min)
              ├── Revisar takes (5 min)
              └── Trocar figurino se necessário

14:45 - 15:15 │ REEL 2
              ├── Mesma estrutura
              └── Ajustar energia/tom conforme tema

15:15 - 15:45 │ REEL 3
              ├── Mesma estrutura
              └── Gravar B-roll extra se sobrar tempo

15:45 - 16:00 │ REVISÃO GERAL
              ├── Assistir todos os takes
              ├── Marcar os melhores
              └── Anotar ajustes para edição
```

### Dia 3: Edição de Reels + Design (3-4 horas)

```
SESSÃO DE EDIÇÃO

FERRAMENTAS RECOMENDADAS:
├── Edição de vídeo: CapCut, InShot, VN
├── Design: Canva, Figma
├── Organização: Notion, Trello
└── Agendamento: Meta Business Suite, Later

ROTEIRO DE EDIÇÃO:

10:00 - 11:00 │ EDIÇÃO REEL 1
              ├── Importar takes selecionados
              ├── Cortar e montar sequência
              ├── Adicionar áudio/música
              ├── Inserir textos na tela
              ├── Aplicar efeitos/transições
              ├── Ajustar cores se necessário
              └── Exportar em alta qualidade

11:00 - 12:00 │ EDIÇÃO REEL 2 e 3
              ├── Repetir processo
              └── Manter consistência visual

12:00 - 12:30 │ PAUSA

12:30 - 14:00 │ DESIGN CARROSSÉIS
              ├── Abrir template no Canva
              ├── Carrossel 1:
              │   ├── Criar Slide 1 (thumbnail)
              │   ├── Preencher slides 2-9
              │   └── Slide 10 (CTA)
              ├── Carrossel 2:
              │   └── Repetir processo
              └── Exportar como PDF ou imagens

14:00 - 15:00 │ DESIGN CARDS ÚNICOS
              ├── Abrir templates de card
              ├── Criar 5 cards:
              │   ├── 2 dicas/valor
              │   ├── 1 citação
              │   ├── 1 engajamento
              │   └── 1 pessoal/bastidores
              └── Manter identidade visual consistente

15:00 - 15:30 │ REVISÃO FINAL
              ├── Verificar ortografia
              ├── Conferir dimensões corretas
              ├── Testar legibilidade
              └── Organizar arquivos por data de postagem
```

### Dia 4: Captions + Agendamento (2 horas)

```
SESSÃO DE FINALIZAÇÃO

16:00 - 17:00 │ ESCREVER CAPTIONS
              ├── Para cada conteúdo:
              │   ├── Hook (linha 1)
              │   ├── Corpo (3-5 parágrafos)
              │   ├── CTA específico
              │   └── Hashtags (10-15)
              ├── Usar template de caption
              └── Revisar tom de voz

17:00 - 17:30 │ PREPARAR HASHTAGS
              ├── Criar sets de hashtags:
              │   ├── Set 1: Reels
              │   ├── Set 2: Carrossel
              │   └── Set 3: Cards
              └── Alternar para evitar shadowban

17:30 - 18:00 │ AGENDAR PUBLICAÇÕES
              ├── Meta Business Suite ou ferramenta preferida
              ├── Distribuir ao longo da semana:
              │   ├── Seg: Card + Reel
              │   ├── Ter: Carrossel
              │   ├── Qua: Card + Stories
              │   ├── Qui: Reel
              │   ├── Sex: Card + Carrossel
              │   ├── Sáb: Reel
              │   └── Dom: Card (leve/pessoal)
              └── Configurar primeira publicação de cada
```

## Checklist de Batch Production

### Antes de Começar

```
PREPARAÇÃO:
[ ] Ideias brainstormadas (20+)
[ ] Tendências pesquisadas
[ ] Roteiros escritos
[ ] Templates preparados
[ ] Equipamento testado
[ ] Ambiente organizado
[ ] Figurinos separados
[ ] Água e snacks disponíveis
```

### Durante a Produção

```
GRAVAÇÃO DE REELS:
[ ] Iluminação adequada?
[ ] Áudio limpo?
[ ] Enquadramento correto?
[ ] Múltiplos takes gravados?
[ ] Energia consistente?
[ ] B-roll capturado?

DESIGN:
[ ] Dimensões corretas (1080x1350)?
[ ] Texto legível?
[ ] Cores da marca?
[ ] Hierarquia visual clara?
[ ] Indicador de swipe (carrossel)?
[ ] CTA visível?
```

### Após a Produção

```
FINALIZAÇÃO:
[ ] Todos os arquivos exportados?
[ ] Nomeados por data de publicação?
[ ] Captions escritas?
[ ] Hashtags preparadas?
[ ] Agendamentos configurados?
[ ] Backup feito?
```

## Dicas de Produtividade

### Otimizando o Tempo

```
1. BATCHING POR ATIVIDADE
   ├── Não misture gravação com edição
   ├── Faça todo o design de uma vez
   └── Escreva todas as captions juntas

2. TEMPLATES REUTILIZÁVEIS
   ├── Criar 5-10 templates de carrossel
   ├── Ter presets de cor/filtro salvos
   └── Manter biblioteca de hooks

3. PREPARAÇÃO NA NOITE ANTERIOR
   ├── Revisar roteiros
   ├── Separar figurinos
   └── Carregar equipamentos

4. BLOQUEIO DE DISTRAÇÕES
   ├── Modo avião durante gravação
   ├── Ambiente silencioso
   └── Timer para cada etapa

5. ENERGIA E FOCO
   ├── Gravar no horário de maior energia
   ├── Pausas a cada 45-60 min
   └── Hidratação constante
```

### Escalando a Produção

```
NÍVEL 1 (Iniciante):
├── 1 Reel + 1 Carrossel + 3 Cards/semana
├── 1 dia de produção
└── Ferramentas gratuitas

NÍVEL 2 (Intermediário):
├── 3 Reels + 2 Carrosséis + 5 Cards/semana
├── 2 dias de produção
└── Algumas ferramentas pagas

NÍVEL 3 (Avançado):
├── 5+ Reels + 3 Carrosséis + 7 Cards/semana
├── 2-3 dias de produção
└── Equipe ou freelancers

NÍVEL 4 (Escala):
├── Conteúdo diário em múltiplos formatos
├── Equipe dedicada
└── Processos documentados
```

## Troubleshooting

### Problemas Comuns

| Problema | Causa | Solução |
|----------|-------|---------|
| Falta de ideias | Sem pesquisa de tendências | Reservar 30min/semana para pesquisa |
| Gravação demorada | Muitos takes | Limitar a 5 takes por cena |
| Edição lenta | Sem template | Criar presets e templates |
| Inconsistência visual | Sem padrão | Definir brand guide |
| Burnout | Produção excessiva | Começar com menos e escalar |
| Baixo engajamento | Hooks fracos | Testar 3-5 variantes de hook |
