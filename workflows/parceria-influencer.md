# 🤝 Workflow: Parceria com Influenciadores

> Workflow do Marketing OS. A numeração e os padrões de orquestração canônicos estão em `skills/marketing-os/SKILL.md` (seção "Padrões de Orquestração").

Processo completo para encontrar, abordar e gerenciar parcerias com influenciadores.

---

## 📊 Visão Geral

| Fase | Duração | Atividades |
|------|---------|------------|
| **1. Estratégia** | 1 semana | Definir objetivos, budget, perfil |
| **2. Prospecção** | 1-2 semanas | Encontrar e analisar influenciadores |
| **3. Abordagem** | 1 semana | Contato inicial e negociação |
| **4. Briefing** | 3-5 dias | Enviar briefing e produto |
| **5. Execução** | 1-2 semanas | Criação e aprovação do conteúdo |
| **6. Análise** | 1 semana | Medir resultados e documentar |

---

## 🎯 FASE 1: ESTRATÉGIA

### 1.1 Definir Objetivos

```
OBJETIVO PRINCIPAL:
□ Awareness (alcance de marca)
□ Engajamento (construir comunidade)
□ Tráfego (visitas para site/LP)
□ Vendas (conversões diretas)
□ UGC (conteúdo para reutilizar)

METAS ESPECÍFICAS:
• Alcance esperado: [número]
• Engajamento esperado: [número]
• Leads/vendas esperadas: [número]
• Conteúdos gerados: [número]
```

### 1.2 Definir Budget

```
BUDGET TOTAL: R$ [valor]

DISTRIBUIÇÃO:
□ Pagamento ao influenciador: [%]
□ Produto/gift: [%]
□ Produção adicional: [%]
□ Impulsionamento: [%]

MODELOS DE PAGAMENTO:
□ Fee fixo
□ Permuta (produto)
□ Comissão por vendas
□ Híbrido (fixo + comissão)
```

### 1.3 Definir Perfil Ideal

```
TAMANHO:
□ Nano (1-10K): Alto engajamento, baixo custo
□ Micro (10-50K): Bom equilíbrio custo x alcance
□ Médio (50-500K): Alcance relevante
□ Macro (500K-1M): Grande alcance
□ Mega (1M+): Massificação

NICHO:
[Lista de nichos relevantes para sua marca]

CRITÉRIOS ESSENCIAIS:
□ Engajamento mínimo: [X%]
□ Audiência do mesmo público-alvo
□ Valores alinhados com a marca
□ Qualidade de conteúdo
□ Histórico de parcerias
□ Autenticidade
```

## 🔍 FASE 2: PROSPECÇÃO

### 2.1 Onde Encontrar

```
FONTES DE PESQUISA:
□ Hashtags relevantes no Instagram/TikTok
□ Menções da sua marca
□ Menções de concorrentes
□ Plataformas de influenciadores
□ Indicações de outros influenciadores
□ Agências de marketing de influência
```

### 2.2 Análise do Influenciador

```
CHECKLIST DE ANÁLISE:

MÉTRICAS:
□ Número de seguidores: [X]
□ Taxa de engajamento: [X%]
  (Likes + Comentários) / Seguidores x 100
□ Crescimento: [tendência]
□ Frequência de posts: [X/semana]

QUALIDADE:
□ Qualidade visual do conteúdo
□ Qualidade da legenda/copy
□ Consistência de estilo
□ Originalidade

AUDIÊNCIA:
□ Faixa etária compatível
□ Gênero compatível
□ Localização compatível
□ Interesses compatíveis
□ Engajamento real (não bots)

RED FLAGS:
⚠️ Engajamento muito baixo (<1%)
⚠️ Engajamento muito alto (>15%) - possível fake
⚠️ Muitos comentários genéricos
⚠️ Spikes de seguidores suspeitos
⚠️ Parcerias com concorrentes diretos
⚠️ Histórico polêmico/problemático
```

### 2.3 Planilha de Prospecção

```
CAMPOS PARA RASTREAR:

| Nome | @ | Seguidores | Eng% | Nicho | Contato | Status | Notas |
|------|---|------------|------|-------|---------|--------|-------|
|      |   |            |      |       |         |        |       |
```

## 📧 FASE 3: ABORDAGEM

### 3.1 Primeiro Contato

```
CANAL DE CONTATO (em ordem de preferência):
1. Email (mais profissional)
2. DM no Instagram
3. Assessoria/agência (se tiver)

ANTES DE ENVIAR:
□ Seguir e engajar com o perfil (1-2 semanas)
□ Pesquisar nome/email correto
□ Verificar se já fez parceria com concorrente
□ Personalizar a mensagem
```

### 3.2 Template de Abordagem

```
ASSUNTO: Parceria [Nome da Marca] x [Nome do Influenciador]

---

Oi [Nome]!

Tudo bem? Sou [Seu nome], [cargo] da [Marca].

Acompanho seu trabalho há um tempo e adoro
como você [elogio genuíno e específico].

A [Marca] é [descrição em 1 frase] e acredito
que nossos públicos têm muito em comum.

Gostaria de propor uma parceria para
[objetivo geral - ex: divulgação do nosso novo produto].

Posso te contar mais detalhes?

Abraço,
[Seu nome]
[Cargo]
[Marca]
[Link para site/redes]

---

VARIAÇÃO PARA DM:

Oi [Nome]! 👋

Sou da [Marca] e adoramos seu conteúdo sobre [tema].

Temos interesse em uma parceria. Posso te mandar
mais detalhes por email?

Qual o melhor contato?
```

### 3.3 Follow-up

```
SE NÃO RESPONDER EM 5-7 DIAS:

"Oi [Nome]! Passando pra ver se recebeu minha
mensagem sobre a parceria com a [Marca].

Ficamos muito interessados em trabalhar com você!

Posso te ligar ou mandar mais informações?"

---

REGRAS DE FOLLOW-UP:
• Máximo 2-3 follow-ups
• Espaçar 5-7 dias entre cada
• Ser breve e direto
• Não ser insistente
```

### 3.4 Negociação

```
INFORMAÇÕES A ALINHAR:

□ Tipo de conteúdo (stories, feed, reels, etc.)
□ Quantidade de posts
□ Prazo de entrega
□ Exclusividade (se houver)
□ Direitos de uso do conteúdo
□ Valor/permuta
□ Forma de pagamento
□ Revisões permitidas
□ Código de desconto/link trackável
```

---

## 📋 FASE 4: BRIEFING

### 4.1 Documento de Briefing

```
BRIEFING COMPLETO:

1. SOBRE A MARCA
   • Quem somos
   • Valores
   • Tom de voz

2. SOBRE O PRODUTO
   • O que é
   • Benefícios principais
   • Diferenciais
   • Preço

3. PÚBLICO-ALVO
   • Quem queremos atingir
   • Dores e desejos

4. ENTREGÁVEIS
   • Formatos (stories, reels, feed)
   • Quantidade
   • Duração mínima
   • Menções obrigatórias (@, #, link)

5. DIRETRIZES
   • O que PODE fazer
   • O que NÃO pode fazer
   • Referências visuais
   • Tom desejado

6. TIMELINE
   • Data de envio do rascunho
   • Prazo para aprovação
   • Data de publicação

7. PRÓXIMOS PASSOS
   • Como enviar o conteúdo
   • Contato para dúvidas
```

### 4.2 Envio do Produto

```
CHECKLIST DE ENVIO:

□ Produto bem embalado
□ Carta/bilhete personalizado
□ Briefing impresso (opcional)
□ Código de desconto único
□ Tracking enviado ao influenciador
□ Confirmar recebimento
```

---

## 🎬 FASE 5: EXECUÇÃO

### 5.1 Acompanhamento

```
TIMELINE DE ACOMPANHAMENTO:

D+0: Produto enviado
D+3: Confirmar recebimento
D+7: Checar andamento
D+10: Receber rascunho para aprovação
D+12: Enviar feedback/aprovação
D+14: Publicação
D+15: Impulsionar (se aplicável)
```

### 5.2 Aprovação de Conteúdo

```
CHECKLIST DE APROVAÇÃO:

□ Menção correta da marca (@)
□ Produto mostrado adequadamente
□ Tom de voz apropriado
□ Link/código funcionando
□ Hashtags corretas (#publi se pago)
□ Qualidade de imagem/vídeo
□ Sem erros de informação
□ Disclosure adequado (quando aplicável)
```

### 5.3 Feedback

```
TEMPLATE DE FEEDBACK:

"Oi [Nome]! Amei o conteúdo! 🙌

Só algumas pequenas sugestões:
1. [Ajuste 1]
2. [Ajuste 2]

De resto, está perfeito!

Pode fazer esses ajustes e publicar até [data]?"

---

OU SE APROVAR:

"Perfeito, [Nome]! Pode publicar!

Assim que sair, me manda o link?

Obrigado(a) pela parceria!"
```

---

## 📊 FASE 6: ANÁLISE

### 6.1 Métricas a Rastrear

```
MÉTRICAS DO POST:
□ Alcance
□ Impressões
□ Likes
□ Comentários
□ Salvamentos
□ Compartilhamentos
□ Views (se vídeo)

MÉTRICAS DE CONVERSÃO:
□ Cliques no link
□ Uso do código de desconto
□ Vendas geradas
□ Novos seguidores
□ Leads captados

CÁLCULOS:
• CPM = (Investimento / Alcance) x 1000
• CPE = Investimento / Engajamentos
• CPA = Investimento / Conversões
• ROI = (Receita - Investimento) / Investimento
```

### 6.2 Relatório Final

```
TEMPLATE DE RELATÓRIO:

INFLUENCIADOR: [Nome]
DATA: [DD/MM/AAAA]
INVESTIMENTO: R$ [valor]

RESULTADOS:
• Alcance: [X]
• Engajamento: [X]
• Cliques: [X]
• Vendas: [X]

MÉTRICAS:
• CPM: R$ [X]
• CPE: R$ [X]
• CPA: R$ [X]
• ROI: [X]%

ANÁLISE:
[Comentários sobre performance]

RECOMENDAÇÃO:
□ Refazer parceria
□ Não refazer
□ Escalar (mais posts)
```

### 6.3 Relacionamento Pós-Parceria

```
AÇÕES DE RELACIONAMENTO:

□ Agradecer publicamente (comentário, repost)
□ Enviar resultado da campanha
□ Manter contato esporádico (likes, comentários)
□ Convidar para próximas campanhas
□ Programa de embaixadores (se aplicável)
□ Enviar lançamentos como gift
```
